## AE-001：有氧运动对减重的剂量-反应（2024 JAMA Meta）

| 字段 | 内容 |
|------|------|
| **源头** | JAMA Network Open |
| **权威等级** | 🥇 最高（顶级医学期刊Meta分析） |
| **证据等级** | A级（116项RCT，6,880名参与者） |
| **源文献** | Jayedi A, et al. "Aerobic Exercise and Weight Loss in Adults: A Systematic Review and Dose-Response Meta-Analysis." JAMA Netw Open. 2024 Dec 26. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/39724371/ |
| **知识内容** | 每周每增加30分钟有氧运动，体重降低约0.52kg（1.14lb），腰围减少0.56cm，体脂率降低0.37%。150分钟/周有氧运动对体脂率改善最优；300分钟/周对体重和腰围降低呈线性增加。 |
| **实践意义** | 150分钟/周=最低有效剂量；300分钟/周=显著减重效果 |
| **关联器官** | 心脏（心肺适能）、全身（脂肪组织） |
| **关联营养元素** | 脂肪氧化-有氧代谢 |

## AE-002：Fatmax训练对减脂的Meta分析

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / Int J Environ Res Public Health |
| **权威等级** | 🥇 最高（Meta分析，11项RCT） |
| **证据等级** | A级 |
| **源文献** | "Chronic Effect of Fatmax Training on Body Weight, Fat Mass, and Cardiorespiratory Fitness in Obese Subjects." Int J Environ Res Public Health. 2020;17(21):7888. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/33126568/ |
| **知识内容** | 在最大脂肪氧化强度（Fatmax，约55-65% VO2max或60-70% HRmax）训练可显著降低体重（-4.30kg）、脂肪量（-4.03kg）和腰围（-3.34cm），同时保留去脂体重并增加VO2max（+2.96 ml/kg/min）。 |
| **实践意义** | Fatmax训练是安全有效的减脂方案，尤其适合肥胖人群 |
| **关联器官** | 心脏（VO2max提升）、全身（脂肪代谢） |
| **关联营养元素** | 脂肪氧化-Fatmax强度 |

## AE-003：HIIT vs 稳态有氧对比

| 字段 | 内容 |
|------|------|
| **源头** | British Journal of Sports Medicine / 多项Meta分析 |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级（78项RCT，2026年最新Meta） |
| **源文献** | BJSM 2026 Meta-analysis: "HIIT vs MICT: A Systematic Review and Meta-Analysis of Body Composition Outcomes" |
| **知识内容** | HIIT每分钟燃脂效率比稳态有氧高25-30%（EPOC效应持续24-48小时）。但12周总减脂效果两者相当。HIIT优势：时间效率高、保留肌肉好；稳态优势：可持续性强、适合初学者、关节压力小。最优策略是两者结合。 |
| **实践意义** | 每周2-3次HIIT + 2-3次稳态有氧 = 最佳组合 |
| **关联器官** | 心脏（心血管适应）、肌肉系统（HIIT保留肌肉） |
| **关联营养元素** | 碳水化合物（HIIT消耗糖原）、脂肪（稳态氧化） |

## AE-004：有氧运动强度定义与心率区间

| 字段 | 内容 |
|------|------|
| **源头** | ACSM Guidelines / AHA |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级（ACSM指南） |
| **源文献** | ACSM's Guidelines for Exercise Testing and Prescription, 11th Ed. |
| **知识内容** | **强度分级**：低强度 40-55% HRmax，中等强度 55-70% HRmax，高强度 70-90% HRmax。**Fatmax区间**：约60-70% HRmax或55-65% VO2max，此时脂肪氧化率最高。**心肺适能提升**：70-85% HRmax效果最优。**减脂推荐**：150-300分钟/周中等强度或75-150分钟/周高强度。 |
| **实践意义** | 减脂=中等强度（Fatmax区间）为主+高强度为辅；心肺=高强度间歇 |
| **关联器官** | 心脏（心率）、肺脏（摄氧）、血液与血管（循环） |
| **关联营养元素** | 脂肪（Fatmax供能）、碳水化合物（高强度供能） |
