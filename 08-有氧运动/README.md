# 08 - 有氧运动

> **Aerobic Exercise / Cardiorespiratory Training**
> 
> 有氧训练、心肺适能、耐力训练、脂肪代谢

## 子主题

- 有氧训练生理适应
- 心肺适能评估
- 中等强度持续训练（MICT）
- 低强度稳态训练（LISS）
- 高强度间歇训练（HIIT）有氧应用
- 心率区间训练
- 脂肪供能区与脂肪最大氧化强度（Fatmax）
- 耐力训练（长跑、骑行、游泳）
- 有氧训练与减脂
- 训练周期化（有氧）
- 有氧训练对心血管健康的影响
- 过度有氧的风险
