## 数据格式

每条知识记录包含：
- **源头标签** - 知识来源机构/期刊
- **权威等级** - 🥇最高 🥈高 🥉中等
- **证据等级** - A级(系统综述/RCT) B级(队列研究) C级(专家共识)
- **源文献名** - 完整引用信息
- **知识内容** - 核心知识点
- **实践意义** - 应用建议
