## O3-001：Omega-3对心血管疾病的一级和二级预防

| 字段 | 内容 |
|------|------|
| **源头** | Cochrane Library |
| **权威等级** | 🥇 最高（Cochrane系统综述） |
| **证据等级** | A级（79项RCT，112,059名参与者） |
| **源文献** | Abdelhamid AS, et al. "Omega-3 fatty acids for the primary and secondary prevention of cardiovascular disease." Cochrane Database Syst Rev. 2018;7(7):CD003177. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/30019766/ |
| **知识内容** | Cochrane系统综述（79项RCT）：增加EPA/DHA摄入对全因死亡率影响很小或没有（RR 0.98），对心血管事件有微弱保护作用（RR 0.97），但降低血清甘油三酯效果明确。增加ALA摄入可能略微降低心血管事件风险。总体证据等级中等。 |
| **关联器官** | 心脏（心血管事件）、血液与血管（甘油三酯/血脂） |
| **关联营养元素** | Omega-3脂肪酸-EPA/DHA/ALA |

## O3-002：Omega-3对心血管疾病预防的Meta分析（2024更新）

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / Cardiovascular Drugs and Therapy |
| **权威等级** | 🥇 最高（系统综述与Meta分析） |
| **证据等级** | A级 |
| **源文献** | Yan J, et al. "Efficacy and Safety of Omega-3 Fatty Acids in the Prevention of Cardiovascular Disease: A Systematic Review and Meta-analysis." Cardiovasc Drugs Ther. 2024;38(4):799-817. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/36103100/ |
| **知识内容** | Omega-3脂肪酸补充显著降低心血管死亡率、心肌梗死和冠心病事件。高剂量EPA（>2g/天）的益处比低剂量更明显。Omega-3安全性良好，无严重不良事件增加。 |
| **实践意义** | 心血管高风险人群可考虑高剂量EPA补充（需医生指导）；普通人群每周2-3次脂肪鱼优先 |
| **关联器官** | 心脏（心肌梗死预防）、血液与血管（冠心病） |
| **关联营养元素** | Omega-3-高剂量EPA |

## O3-003：Omega-3与大脑/神经保护

| 字段 | 内容 |
|------|------|
| **源头** | ISSN / 多篇系统综述 |
| **权威等级** | 🥈 高 |
| **证据等级** | B级 |
| **源文献** | 多篇研究综合 |
| **知识内容** | DHA占大脑灰质脂肪酸的约30%。观察性研究表明，高鱼类摄入与认知衰退风险降低和阿尔茨海默病风险降低相关。RCT证据对已患痴呆者的治疗效果较弱，但预防作用有一定支持。孕期DHA补充对胎儿神经发育有益。 |
| **实践意义** | 孕期和老年人应确保DHA摄入充足（≥250mg/天EPA+DHA） |
| **关联器官** | 大脑与神经系统（认知保护、神经发育） |
| **关联营养元素** | Omega-3-DHA |

## 知识源交叉验证

✅ Cochrane + 2024 Meta分析 + AHA建议 → Omega-3对心血管有益，尤其高剂量EPA
✅ 观察性研究一致 → 脂肪鱼摄入与心血管/认知健康正相关
⚠️ 低剂量补充剂效果存争议 → 食物来源优先原则
