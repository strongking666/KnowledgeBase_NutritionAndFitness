## DIS-001：2型糖尿病缓解——DiRECT试验的低热量饮食策略

| 字段 | 内容 |
|------|------|
| **源头** | The Lancet / The Lancet Diabetes & Endocrinology |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级（RCT） |
| **源文献** | Lean ME, et al. "Primary care-led weight management for remission of type 2 diabetes (DiRECT): an open-label, cluster-randomised trial." Lancet. 2018;391(10120):541-551. + Lean ME, et al. "Durability of a primary care-led weight-management intervention for remission of type 2 diabetes: 2-year results of the DiRECT open-label, cluster-randomised trial." Lancet Diabetes Endocrinol. 2019;7(5):344-355. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/29221645/ |
| **知识内容** | DiRECT试验是2型糖尿病(T2DM)缓解的里程碑RCT：①干预组使用完全液体配方低热量饮食（825-853 kcal/d × 3-5个月），然后逐步恢复食物+维持支持；②1年时干预组46%达到糖尿病缓解（HbA1c<6.5%且停用降糖药），对照组仅4%；③2年时缓解率36%（减重≥10 kg者缓解率64%）；④5年随访（2024）仍有一定比例维持缓解。机制：极低热量→快速减重→肝脏和胰腺脂肪减少→β细胞功能恢复。"双循环假说"：肝脏脂肪↓→胰岛素敏感性↑→胰腺脂肪↓→胰岛素分泌恢复。 |
| **实践意义** | T2DM可通过显著减重（>10-15 kg）实现缓解，低热量饮食为证据最强路径 |
| **关联器官** | 肝脏（脂肪异位沉积）、胰腺（β细胞功能） |
| **关联营养元素** | 总热量限制、碳水化合物（并非必须极低碳水） |

## DIS-002：痛风与高尿酸——嘌呤、果糖与酒精的饮食管理

| 字段 | 内容 |
|------|------|
| **源头** | NEJM / Arthritis & Rheumatology / BMJ |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | Choi HK, et al. "Purine-rich foods, dairy and protein intake, and the risk of gout in men." N Engl J Med. 2004;350:1093-1103. + Choi HK, Curhan G. "Soft drinks, fructose consumption, and the risk of gout in men: prospective cohort study." BMJ. 2008;336:309-312. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/15014182/ |
| **知识内容** | 痛风饮食管理的三层策略：①减少嘌呤来源——红肉（风险↑41%/日份）、海鲜（↑51%），但植物嘌呤（豆类、蔬菜）与痛风无关（去污名化）；②果糖是关键驱动因素——含糖饮料每日2+份→痛风风险↑85%，因果糖代谢消耗ATP→产生尿酸；③酒精——啤酒（嘌呤+酒精双重打击）风险最高（↑49%/日杯），烈酒↑15%，红酒无显著关联。保护因素：低脂乳制品（每日2份→风险↓40%+）、樱桃/樱桃汁、咖啡、维生素C。减重是最重要的非药物干预。 |
| **实践意义** | 痛风饮食核心：减少动物嘌呤+果糖+啤酒，增加低脂乳制品和蔬菜，减重 |
| **关联器官** | 肾脏（尿酸排泄）、关节（尿酸结晶沉积） |
| **关联营养元素** | 嘌呤、果糖、维生素C（促进尿酸排泄）、酒精 |

## DIS-003：炎症性肠病(IBD)的排除饮食法——CDED和EEN证据

| 字段 | 内容 |
|------|------|
| **源头** | Gastroenterology / The Lancet Gastroenterology & Hepatology |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级（RCT证据） |
| **源文献** | Levine A, et al. "Crohn's Disease Exclusion Diet Plus Partial Enteral Nutrition Induces Sustained Remission in a Randomized Controlled Trial." Gastroenterology. 2019;157(2):440-450.e8. + Narula N, et al. "Enteral nutritional therapy for induction of remission in Crohn's disease." Cochrane Database Syst Rev. 2018;4:CD000542. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/31170412/ |
| **知识内容** | IBD饮食治疗的两大循证策略：①全肠内营养(EEN)——仅摄入配方液体营养4-6周，儿童克罗恩病缓解率60-80%（与皮质类固醇相当但无副作用），机制可能是排除促炎食物成分+菌群改变；②克罗恩病排除饮食(CDED)——无需完全禁食，排除加工食品、麸质、乳制品、动物脂肪、乳化剂等，配合部分肠内营养(PEN)，6周缓解率68%（与EEN相当但依从性更好）。CDED允许鸡肉、鸡蛋、米饭、土豆、香蕉、苹果等"安全食物"。UC的排除饮食证据弱于CD。 |
| **实践意义** | IBD饮食治疗（EEN/CDED）是药物之外的有效选择，CD证据强于UC |
| **关联器官** | 肠道（黏膜炎症、菌群） |
| **关联营养元素** | 麸质（部分患者）、乳化剂（促炎）、短链脂肪酸 |

## DIS-004：PCOS（多囊卵巢）的饮食干预——低GI和抗炎饮食

| 字段 | 内容 |
|------|------|
| **源头** | Human Reproduction Update / Journal of the Academy of Nutrition and Dietetics |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级（系统综述与Meta分析） |
| **源文献** | Moran LJ, et al. "Dietary composition in the treatment of polycystic ovary syndrome: a systematic review to inform evidence-based guidelines." J Acad Nutr Diet. 2013;113(4):520-545. + Teede HJ, et al. "Recommendations from the 2023 international evidence-based guideline for the assessment and management of polycystic ovary syndrome." Hum Reprod Update. 2023;29(6):645-666. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/23420000/ |
| **知识内容** | 2023国际PCOS循证指南饮食推荐：①核心目标是减重5-10%（无论何种饮食模式，依从性是关键），可改善排卵、胰岛素抵抗和雄激素水平；②低GI饮食可能改善胰岛素敏感性（效应量中等）；③抗炎饮食（地中海模式）可能有益但证据不如减重充分；④无证据支持某一种特定饮食（生酮、无麸质等）优于其他——依从性>组成。肌醇（肌醇与D-手性肌醇40:1比例）有Meta分析支持改善排卵和代谢。关键：PCOS饮食管理应个体化、可持续、以减重和代谢改善为目标。 |
| **实践意义** | PCOS饮食核心是减重5-10%，低GI和地中海模式有额外获益 |
| **关联器官** | 卵巢（排卵功能）、胰腺（胰岛素抵抗）、脂肪组织 |
| **关联营养元素** | 肌醇、低GI碳水化合物、omega-3（抗炎） |

## 知识源交叉验证

✅ T2DM缓解有DiRECT试验A级证据→减重是核心
✅ 痛风饮食管理需关注果糖+啤酒+动物嘌呤，植物嘌呤安全
✅ IBD饮食治疗(CDED/EEN)在CD中证据充分，UC证据较弱
✅ PCOS饮食核心是减重+依从性，低GI有益但非必须
