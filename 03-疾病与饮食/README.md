# 03 - 疾病与饮食

> **Diet-Related Diseases**
> 
> 饮食和生活方式导致的常见疾病：肥胖、糖尿病、心血管病、代谢综合征等

## 子主题

- 肥胖症（病因、评估、管理）
- 2型糖尿病（饮食管理、血糖控制）
- 心血管疾病（饮食预防、血脂管理）
- 代谢综合征
- 高血压（DASH饮食、钠摄入）
- 高脂血症
- 非酒精性脂肪肝（NAFLD）
- 胰岛素抵抗
- 炎症与饮食
- 营养不良（不足与过剩）
- 饮食失调（厌食、贪食、暴食）
- 癌症与饮食关系

## 主要知识源

- WHO Global Report on Diabetes
- ADA Standards of Care
- AHA Dietary Guidelines
- NICE Clinical Guidelines
- Cochrane Reviews on diet-disease
- NEJM / Lancet landmark studies
