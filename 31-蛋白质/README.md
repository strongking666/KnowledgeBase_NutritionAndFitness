# 蛋白质 (蛋白质)

> **维度**: 营养物质/元素
> **关联知识源标签**: 营养-蛋白质

## 常见来源及推荐排序

按补充效果从高到低排序：

1. 乳清蛋白
2. 酪蛋白
3. 鸡胸肉
4. 鸡蛋
5. 牛肉
6. 三文鱼
7. 大豆/豆腐
8. 藜麦
9. 希腊酸奶
10. 蛋白粉

## 缺乏或过量影响

- 缺乏: 肌少症、免疫功能下降、水肿、生长发育迟缓、头发稀疏
- 过量: 肾脏负担增加(已有肾病者)、钙流失(高蛋白饮食)、脱水风险

## 推荐家常菜系及做法

- 鸡胸肉沙拉
- 水煮蛋
- 清蒸鱼
- 豆腐炖汤
- 牛肉炒西兰花
- 藜麦饭
- 希腊酸奶配坚果
