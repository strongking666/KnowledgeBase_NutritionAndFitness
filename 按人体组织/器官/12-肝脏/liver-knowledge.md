## LVR-001：非酒精性脂肪肝(NAFLD)的全球流行与运动干预

| 字段 | 内容 |
|------|------|
| **来源** | Hepatology / Lancet / 多项Meta分析 |
| **权威等级** | 🔬 最高 |
| **证据等级** | A级（流行病学+多项RCT） |
| **源文献** | Younossi ZM, et al. "Global epidemiology of NAFLD." Hepatology. 2023. + Hashida R, et al. "Exercise and NAFLD." J Hepatol. 2022. |
| **知识内容** | 全球NAFLD患病率约25-30%，中国约29%。运动干预独立于减重即可减少肝脏脂肪：每周150分钟中高强度运动可减少肝内脂肪20-30%（MRI测定），即使体重未变。机制：AMPK通路激活→增加脂肪酸氧化→减少肝脏脂质合成。有氧+抗阻联合训练效果优于单一模式。体重减轻7-10%可逆转NASH纤维化。 |
| **实践意义** | 每周≥150分钟有氧+2次抗阻训练是NAFLD一线非药物治疗 |
| **关联器官** | 肝脏（脂肪代谢）、肌肉系统（能量消耗） |
| **关联营养元素** | Omega-3脂肪酸、膳食纤维 |

## LVR-002：酒精代谢与肝脏损伤的阈值

| 字段 | 内容 |
|------|------|
| **来源** | WHO / IARC / Lancet |
| **权威等级** | 🔬 最高 |
| **证据等级** | A级（大规模队列研究） |
| **源文献** | GBD 2020 Alcohol Collaborators. Lancet. 2022. + Rehm J, et al. "Alcohol use and liver disease." J Hepatol. 2023. |
| **知识内容** | 安全饮酒量的科学共识正在下移。GBD 2020：对15-49岁人群，零酒精是最小化健康风险的选择。肝损伤阈值：男性>30g/天、女性>20g/天持续5年以上，酒精性肝病风险显著增加。30g≈啤酒750ml/红酒300ml/白酒75ml。但个体差异极大：约20%重度饮酒者不发生肝硬化（遗传保护），部分人即使低剂量也出现脂肪肝。 |
| **实践意义** | 不存在"安全"饮酒量。若饮酒，男性<25g/天、女性<15g/天，每周至少2天无酒精 |
| **关联器官** | 肝脏（酒精代谢）、大脑与神经系统（神经毒性） |
| **关联营养元素** | 维生素B族（酒精消耗B1/B6/B12） |

## LVR-003：水飞蓟素（奶蓟草）对肝脏保护作用的证据

| 字段 | 内容 |
|------|------|
| **来源** | Cochrane / Hepatology |
| **权威等级** | 🔬 高 |
| **证据等级** | B级（多项RCT，但结果不一致） |
| **源文献** | Rambaldi A, et al. "Milk thistle for alcoholic and/or hepatitis B or C liver diseases." Cochrane Review. 2007. Updated 2022. + Fried MW, et al. Hepatology. 2012. |
| **知识内容** | 水飞蓟素（Silymarin）在体外和动物模型中显示抗氧化、抗炎、抗纤维化作用。但临床证据喜忧参半：对酒精性肝病的死亡率无显著改善（Cochrane）；对NASH患者的转氨酶有中度降低作用（ALT降低约10-15 IU/L）；对药物性肝损伤有一定保护作用（如对乙酰氨基酚过量）。安全性良好，副作用罕见。 |
| **实践意义** | 可作为辅助保护（140mg×3次/天），但不能替代戒酒和生活方式干预 |
| **关联器官** | 肝脏（肝细胞保护） |
| **关联营养元素** | 抗氧化物质 |

## LVR-004：高果糖摄入与肝脏脂肪合成的关联

| 字段 | 内容 |
|------|------|
| **来源** | Journal of Hepatology / AJCN |
| **权威等级** | 🔬 最高 |
| **证据等级** | A级（机制研究+多项RCT） |
| **源文献** | Softic S, et al. "Fructose and hepatic insulin resistance." J Hepatol. 2020. + Stanhope KL, et al. J Clin Invest. 2009. |
| **知识内容** | 果糖不同于葡萄糖：①绕过磷酸果糖激酶调节，不受胰岛素反馈抑制，持续进行脂质新生（DNL）；②主要在肝脏代谢，过量直接转化为甘油三酯；③不刺激胰岛素和瘦素分泌，饱腹感差。含糖饮料（HFCS）是最大来源。每日>50g果糖（≈1L可乐）持续2周即可检测到肝脏脂肪增加。 |
| **实践意义** | 限制添加糖<25g/天。避免含糖饮料。水果中的果糖因纤维缓冲而风险较低 |
| **关联器官** | 肝脏（脂质新生、脂肪肝） |
| **关联营养元素** | 碳水化合物（果糖代谢） |

## LVR-005：蛋白质摄入与肝脏修复

| 字段 | 内容 |
|------|------|
| **来源** | Hepatology / ESPEN Guidelines |
| **权威等级** | 🔬 高 |
| **证据等级** | B级（临床指南+观察性研究） |
| **源文献** | Plauth M, et al. "ESPEN guideline on clinical nutrition in liver disease." Clin Nutr. 2019. + 多项临床研究 |
| **知识内容** | 肝病患者（尤其是肝硬化）处于高代谢状态，蛋白质需求增加：推荐1.2-1.5g/kg/天（健康人为0.8-1.0g）。支链氨基酸(BCAA)在肝性脑病中有特殊作用：可改善轻微肝性脑病症状。但严重肝衰竭需限制蛋白质以防氨中毒（现已放宽，不再是严格禁忌）。夜间加餐（睡前50g碳水化合物+20g蛋白质）可改善肝硬化患者氮平衡。 |
| **实践意义** | 肝病患者应保证优质蛋白摄入（鱼、鸡、豆制品），除非医生因脑病限制 |
| **关联器官** | 肝脏（蛋白质合成、解毒）、肌肉系统（BCAA代谢） |
| **关联营养元素** | 蛋白质、支链氨基酸 |
