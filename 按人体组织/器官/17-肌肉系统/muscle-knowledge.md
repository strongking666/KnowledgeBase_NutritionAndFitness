## MSC-001：蛋白质时机（合成代谢窗口）对肌肉蛋白合成的证据更新

| 字段 | 内容 |
|------|------|
| **源头** | Journal of the International Society of Sports Nutrition (JISSN) |
| **权威等级** | 🥇 最高（多项Meta分析） |
| **证据等级** | A级 |
| **源文献** | Schoenfeld BJ, Aragon AA, Krieger JW. "The effect of protein timing on muscle strength and hypertrophy: a meta-analysis." JISSN. 2013; + 后续更新研究 |
| **知识内容** | 传统"合成代谢窗口"（训练后30-60分钟内必须摄入蛋白质）的观念已被证据更新：①Meta分析显示蛋白质总摄入量和每餐分布比时机更重要；②训练后窗口并非30分钟而是可延长至4-6小时（尤其训练前已摄入蛋白质者）；③训练前蛋白质摄入与训练后摄入对肌肉蛋白合成的效果相近；④例外情况：空腹训练者或每日蛋白质总量不足者，训练后及时补充更为关键。总体结论：总蛋白量 > 分布 > 时机。 |
| **实践意义** | 优先保证每日总蛋白量（1.6-2.2g/kg），每3-4小时分布，不必纠结"黄金30分钟" |
| **关联器官** | 肌肉系统（肌肉蛋白合成） |
| **关联营养元素** | 蛋白质（总量与分布）、亮氨酸（触发因子） |

## MSC-002：亮氨酸阈值理论——每餐蛋白质最低剂量

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / JISSN / American Journal of Clinical Nutrition |
| **权威等级** | 🥇 最高（多项机制研究与临床实验） |
| **证据等级** | A级（机制明确+临床验证） |
| **源文献** | 多项研究综合，包括Norton LE, Layman DK, et al. + Paddon-Jones D, et al. |
| **知识内容** | 亮氨酸阈值理论指出：每餐蛋白质必须达到约2.5-3.0g亮氨酸（约25-35g高质量蛋白质）才能充分激活mTOR通路，触发肌肉蛋白合成。低于此阈值的蛋白质摄入效率显著降低，因为亮氨酸作为mTOR的别构激活剂需要达到临界浓度。关键细节：①植物蛋白亮氨酸含量较低，需要更大总量达到阈值；②老年人存在"合成代谢抵抗"，阈值升高至约3.5-4.0g亮氨酸；③乳清蛋白亮氨酸含量最高（约10-12%），最易达到阈值。 |
| **实践意义** | 每餐≥30g高质量蛋白质（或亮氨酸2.5-3g），老年人需更高 |
| **关联器官** | 肌肉系统（mTOR信号通路）、消化系统（蛋白质消化吸收） |
| **关联营养元素** | 亮氨酸（mTOR激活）、蛋白质（必需氨基酸谱） |

## MSC-003：肌酸对肌肉力量和体积的增肌效果

| 字段 | 内容 |
|------|------|
| **源头** | JISSN / 多项Meta分析 |
| **权威等级** | 🥇 最高（数百项RCT综合） |
| **证据等级** | A级（证据极其丰富） |
| **源文献** | 多项Meta分析综合，包括Branch JD, et al. + Kreider RB, et al. JISSN立场声明 |
| **知识内容** | 肌酸是研究最透彻、效果最确切的合法增补剂：①一水肌酸补充可使最大力量增加约5-15%，肌肉体积增加约1-2kg（部分为细胞水合）；②效果在短时高强度运动中最显著（<30秒爆发力），耐力运动获益有限；③负荷期（20g/d×5-7天）+维持期（3-5g/d）可加速饱和，也可直接3-5g/d持续4周达到同等效果；④约20-30%人群对肌酸反应较弱（饮食中已摄入较高量肉类者）；⑤长期使用安全性良好（肾脏健康者）。 |
| **实践意义** | 肌酸是最有效的合法增补剂之一，适合抗阻训练和爆发力运动 |
| **关联器官** | 肌肉系统（磷酸肌酸系统）、肾脏（代谢与排泄） |
| **关联营养元素** | 肌酸（一水肌酸）、蛋白质（协同增肌） |

## MSC-004：年龄相关肌肉流失（少肌症）的营养和运动对策

| 字段 | 内容 |
|------|------|
| **源头** | The Lancet / JAMA / ESPEN指南 |
| **权威等级** | 🥇 最高（全球共识指南） |
| **证据等级** | A级 |
| **源文献** | Cruz-Jentoft AJ, et al. "Sarcopenia: revised European consensus on definition and diagnosis." Age Ageing. 2019. + ESPEN指南 |
| **知识内容** | 少肌症（Sarcopenia）始于约40岁，50岁后肌肉量每年减少约1-2%，力量下降更快（约3%）。核心对策：①抗阻训练是预防和逆转少肌症最有效的干预（效果>营养补充）；②蛋白质需求随年龄增加（1.2-1.5g/kg vs 年轻人0.8g/kg），老年人每餐需≥35g蛋白质克服"合成代谢抵抗"；③亮氨酸补充（3-4g/餐）+维生素D（≥800 IU/d）协同增效；④ω-3脂肪酸（约2g/d）可改善肌肉蛋白合成敏感性。综合干预（运动+营养）效果远优于单一措施。 |
| **实践意义** | 40岁后开始抗阻训练+增加蛋白质摄入，越早越好 |
| **关联器官** | 肌肉系统（肌纤维萎缩）、骨骼与关节（肌骨单位）、内分泌系统（睾酮/生长激素下降） |
| **关联营养元素** | 蛋白质（每餐≥35g）、亮氨酸、维生素D、ω-3脂肪酸 |

## MSC-005：睡眠对肌肉恢复和睾酮/皮质醇比的影响

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / Sleep Medicine Reviews |
| **权威等级** | 🥈 高（多项对照研究） |
| **证据等级** | B级（机制明确，因果研究中等） |
| **源文献** | 多项研究综合，包括Leproult R, Van Cauter E. JAMA. 2011; + Knowles OE, et al. |
| **知识内容** | 睡眠不足严重损害肌肉恢复：①每晚睡5小时（vs 8小时）持续1周，睾酮水平下降约10-15%，皮质醇升高约20%，睾酮/皮质醇比（合成/分解标志）显著恶化；②慢波睡眠（深度睡眠）阶段是生长激素脉冲分泌的高峰期，睡眠剥夺直接减少GH分泌；③睡眠不足抑制肌肉蛋白合成，加速蛋白质分解；④恢复性睡眠（补觉）可部分逆转激素紊乱，但肌肉蛋白合成抑制可能持续更久。④运动后睡眠对组织修复、糖原再填充和神经恢复都不可替代。 |
| **实践意义** | 睡眠不足时增肌效果大打折扣，7-9小时睡眠是增肌的基础条件 |
| **关联器官** | 肌肉系统（恢复与合成）、内分泌系统（睾酮/皮质醇/GH轴）、神经系统（恢复） |
| **关联营养元素** | 镁（睡眠质量）、锌（睾酮合成）、蛋白质（睡前缓释蛋白） |
