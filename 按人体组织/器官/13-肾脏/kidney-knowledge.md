## KDN-001：蛋白质摄入与肾功能——高蛋白饮食的安全性

| 字段 | 内容 |
|------|------|
| **来源** | JASN / AJKD / 多项Meta分析 |
| **权威等级** | 🔬 最高 |
| **证据等级** | A级（多项RCT+队列研究） |
| **源文献** | Devries MC, et al. "Protein and kidney function." J Nutr. 2018. + Ko GJ, et al. "Dietary protein intake and CKD." JASN. 2020. |
| **知识内容** | 对于健康肾脏，高蛋白饮食（2-3g/kg/天）不会损害肾功能。但已有CKD（eGFR<60）者，高蛋白摄入会加速肾功能下降。机制：蛋白质代谢产生尿素和酸负荷，增加肾小球滤过率（肾小球高滤过）。关键区分：健康人→安全；CKD 3-5期→限制蛋白质至0.6-0.8g/kg/天。植物蛋白比动物蛋白对肾脏更友好（酸负荷更低）。 |
| **实践意义** | 健康人健身高蛋白饮食安全；每年体检查肌酐和eGFR；CKD患者遵医嘱限蛋白 |
| **关联器官** | 肾脏（肾小球滤过）、肌肉系统（肌酐产生） |
| **关联营养元素** | 蛋白质、钠、钾、磷 |

## KDN-002：每日饮水量对肾结石和CKD的影响

| 字段 | 内容 |
|------|------|
| **来源** | NEJM / CJASN |
| **权威等级** | 🔬 最高 |
| **证据等级** | A级（RCT+大型队列） |
| **源文献** | Clark WF, et al. "Effect of Coaching to Increase Water Intake on Kidney Function Decline." JAMA. 2018. + Borghi L, et al. "Urinary volume, water and recurrences in idiopathic calcium nephrolithiasis." J Urol. 1996. |
| **知识内容** | 增加饮水量使尿量>2L/天可降低肾结石复发率约50%（经典RCT，5年随访）。对CKD 3期患者，每日额外饮水1-1.5L未显著减缓肾功能下降（JAMA 2018），但对蛋白尿有轻微改善。理想尿量：2-2.5L/天（对应饮水量约2.5-3L）。 |
| **实践意义** | 肾结石病史者饮水量>2.5L/天；一般人饮水量1.5-2L/天足够 |
| **关联器官** | 肾脏（尿液稀释、结石形成） |
| **关联营养元素** | 水、钠、钙 |

## KDN-003：钠摄入与慢性肾病进展

| 字段 | 内容 |
|------|------|
| **来源** | Kidney International / Cochrane |
| **权威等级** | 🔬 最高 |
| **证据等级** | A级（多项RCT+Meta分析） |
| **源文献** | McMahon EJ, et al. "Altered dietary salt intake for people with CKD." Cochrane Review. 2021. + Garofalo C, et al. Kidney Int. 2018. |
| **知识内容** | 低钠饮食（<2g钠/天≈<5g盐/天）对CKD患者：降低血压5-8mmHg（收缩压）、减少蛋白尿约25-30%、增强ACEI/ARB类药物的降蛋白尿效果。对于透析患者，限制钠摄入可减少透析间期体重增加和口渴感。但对硬终点（进入透析/死亡）的RCT证据仍不足。 |
| **实践意义** | CKD患者限钠至<2g/天。用香料替代盐、避免加工食品 |
| **关联器官** | 肾脏（钠排泄、血压调节）、心脏（容量负荷） |
| **关联营养元素** | 钠、钾 |

## KDN-004：钾管理在慢性肾病中的双面性

| 字段 | 内容 |
|------|------|
| **来源** | AJKD / NDT |
| **权威等级** | 🔬 高 |
| **证据等级** | B级（观察性研究+机制研究） |
| **源文献** | Kovesdy CP, et al. "Potassium homeostasis in health and disease." NDT. 2022. + 多项队列研究 |
| **知识内容** | CKD早期（eGFR>45）：高钾摄入有益（降低血压、心血管风险），植物性食物（高钾）对肾脏有保护作用。CKD晚期（eGFR<30）或使用RAAS阻断剂：需限制钾摄入（<2-3g/天），因高钾血症风险增加。矛盾：植物性高钾食物同时提供碱化作用（有益），但需谨慎控制量。透析患者通常需严格限制钾。 |
| **实践意义** | CKD分期决定钾策略：早期多摄入，晚期限制。定期查血钾 |
| **关联器官** | 肾脏（钾排泄）、心脏（血钾与心律） |
| **关联营养元素** | 钾、钠 |

## KDN-005：运动训练对CKD患者的获益

| 字段 | 内容 |
|------|------|
| **来源** | AJKD / CJASN / 多项RCT综合 |
| **权威等级** | 🔬 最高 |
| **证据等级** | A级（多项RCT+系统综述） |
| **源文献** | Heiwe S, Jacobson SH. "Exercise training for adults with CKD." Cochrane Review. 2014. Updated 2021. + 多项RCT |
| **知识内容** | 规律运动对CKD 3-4期患者：提高VO2peak 15-20%、改善6分钟步行距离30-50米、降低血压5-8mmHg、改善生活质量评分。有氧+抗阻联合训练效果最佳。对透析患者：透析中运动（脚踏车）可行且有效。运动不加速肾功能下降。推荐：每周3-5次，每次30-60分钟，中等强度。 |
| **实践意义** | CKD不是运动禁忌。每周150分钟中等强度运动安全且有益 |
| **关联器官** | 肾脏、心脏、肌肉系统 |
| **关联营养元素** | 蛋白质（运动后修复） |
