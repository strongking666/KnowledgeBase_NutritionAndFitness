## HRT-001：有氧运动对心脏健康的剂量-效应关系

| 字段 | 内容 |
|------|------|
| **来源** | American Heart Association / 多项Meta分析 |
| **权威等级** | 🔬 最高 |
| **证据等级** | A级 |
| **源文献** | AHA Scientific Statement. "Physical Activity and Cardiovascular Health." Circulation. 2023. + 多项RCT综合 |
| **知识内容** | 每周150-300分钟中等强度有氧运动可使心血管疾病风险降低20-30%。剂量效应：每增加500 MET-min/周，冠心病风险降低约10-15%。高强度间歇训练(HIIT)对心肺功能的改善效率高于等时中等强度持续训练。但超过500 MET-min/周后边际收益递减，且可能增加房颤风险。 |
| **实践意义** | 每周5天×30分钟中等强度（心率120-140）或3天×25分钟高强度即可达到心脏保护阈值 |
| **关联器官** | 心脏（心肌重塑、心率变异性） |
| **关联营养元素** | Omega-3脂肪酸、辅酶Q10、钾 |

## HRT-002：Omega-3脂肪酸降低心血管事件的证据

| 字段 | 内容 |
|------|------|
| **来源** | NEJM / JAMA / Cochrane |
| **权威等级** | 🔬 最高 |
| **证据等级** | A级（多中心RCT综合） |
| **源文献** | Bhatt DL, et al. "REDUCE-IT Trial." N Engl J Med. 2019;380:11-22. + Manson JE, et al. "VITAL Trial." N Engl J Med. 2019;380:23-32. |
| **知识内容** | REDUCE-IT试验：高剂量EPA（4g/天）使高危患者心血管事件降低25%（NNT=21）。但VITAL试验：普通剂量（1g/天）对一级预防无显著效果。关键差异在于剂量和人群基线风险。EPA而非DHA是主要获益成分。机制：降低甘油三酯（20-30%）、抗炎、稳定斑块。 |
| **实践意义** | 已确诊CVD者：EPA 2-4g/天有明确获益。健康人群：食补优先（每周2-3次深海鱼） |
| **关联器官** | 心脏（心血管事件）、血液与血管（甘油三酯） |
| **关联营养元素** | Omega-3脂肪酸（EPA/DHA） |

## HRT-003：钠钾平衡与血压控制

| 字段 | 内容 |
|------|------|
| **来源** | WHO / NEJM / Cochrane |
| **权威等级** | 🔬 最高 |
| **证据等级** | A级（多中心RCT综合） |
| **源文献** | He FJ, MacGregor GA. "Effect of longer-term modest salt reduction on blood pressure." Cochrane Review. 2013. + Mente A, et al. "PURE Study." Lancet. 2018. |
| **知识内容** | 钠摄入每降低1g/天，收缩压降低约2-5mmHg（高血压者更显著）。但钠摄入与心血管事件的J型曲线存在争议：PURE研究发现钠摄入<3g/天和>5g/天均增加风险。钾摄入每增加1g/天可降低血压1-2mmHg。钠钾比是比单一指标更好的预测因子，理想比值<1。 |
| **实践意义** | 目标：钠<5g/天，钾>3.5g/天。增加蔬菜水果摄入是最简单的钾补充方式 |
| **关联器官** | 心脏（血压负荷）、肾脏（钠排泄）、血液与血管（血压调节） |
| **关联营养元素** | 钠、钾 |

## HRT-004：运动训练对心肌重塑的正面效应（运动员心脏）

| 字段 | 内容 |
|------|------|
| **来源** | JACC / European Heart Journal |
| **权威等级** | 🔬 最高 |
| **证据等级** | A级（多项纵向研究） |
| **源文献** | Pelliccia A, et al. "Athlete's Heart." JACC. 2018. + 多项运动员队列研究 |
| **知识内容** | 长期耐力训练导致心脏向心性肥厚（左室壁增厚+腔室增大），是生理性适应而非病理性。运动员心脏特征：左室质量增加10-20%，每搏输出量增加15-25%，静息心率降低（40-60bpm）。与病理性肥厚的区别：舒张功能正常、无纤维化、停止训练可逆转。力量训练主要引起向心性肥厚（壁厚增加>腔室增大）。 |
| **实践意义** | 运动员心脏是良性适应，不需要治疗。体检发现左室肥厚时应结合运动史判断 |
| **关联器官** | 心脏（心肌重塑）、肌肉系统（运动刺激） |
| **关联营养元素** | 蛋白质（心肌修复） |

## HRT-005：辅酶Q10在心力衰竭中的作用

| 字段 | 内容 |
|------|------|
| **来源** | JACC: Heart Failure / Cochrane |
| **权威等级** | 🔬 最高 |
| **证据等级** | B级（Q-SYMBIO RCT + 多项小规模试验） |
| **源文献** | Mortensen SA, et al. "Q-SYMBIO Study." JACC Heart Fail. 2014;2(6):641-649. |
| **知识内容** | Q-SYMBIO RCT（420名心衰患者，随访2年）：辅酶Q10 100mg×3次/天使主要不良心血管事件降低43%（HR=0.57），心血管死亡降低43%，全因死亡降低42%。机制：改善心肌细胞线粒体能量代谢、抗氧化。注意：他汀类药物会降低内源性辅酶Q10水平。但需要更多大样本研究验证。 |
| **实践意义** | 心衰患者可考虑补充辅酶Q10（100-300mg/天），尤其使用他汀类药物者 |
| **关联器官** | 心脏（心肌能量代谢） |
| **关联营养元素** | 辅酶Q10、抗氧化物质 |
