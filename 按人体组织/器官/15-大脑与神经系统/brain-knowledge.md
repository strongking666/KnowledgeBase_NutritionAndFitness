## BRN-001：有氧运动与脑源性神经营养因子(BDNF)的关联

| 字段 | 内容 |
|------|------|
| **来源** | Nature Reviews Neuroscience / 多项RCT综合 |
| **权威等级** | 🔬 最高 |
| **证据等级** | A级（多项RCT+机制研究） |
| **源文献** | Erickson KI, et al. "Exercise training increases size of hippocampus and improves memory." PNAS. 2011. + Szuhany KL, et al. J Psychiatr Res. 2015. |
| **知识内容** | 急性运动后BDNF升高32%（效应量d=0.55，Meta分析）。规律有氧运动（6个月以上）可增加海马体积约2%（相当于逆转1-2年的年龄相关萎缩）。机制：运动→骨骼肌分泌组织蛋白酶B和鸢尾素→穿过血脑屏障→上调BDNF→促进神经发生和突触可塑性。高强度间歇运动(HIIT)对BDNF的提升可能优于等时中等强度运动。 |
| **实践意义** | 每周≥150分钟有氧运动是保护大脑最有效的生活方式干预 |
| **关联器官** | 大脑与神经系统（神经发生、海马可塑性）、肌肉系统（肌因子分泌） |
| **关联营养元素** | Omega-3脂肪酸（BDNF协同）、咖啡因（急性认知增强） |

## BRN-002：Omega-3脂肪酸与认知衰退的预防

| 字段 | 内容 |
|------|------|
| **来源** | JAMA / Neurology / 多项Meta分析 |
| **权威等级** | 🔬 最高 |
| **证据等级** | B级（观察性证据强，RCT结果不一致） |
| **源文献** | Yurko-Mauro K, et al. "DHA and cognitive decline." Alzheimer's Dement. 2010. + Chew EY, et al. "AREDS2." JAMA. 2015. |
| **知识内容** | 观察性研究一致显示：高鱼类摄入（每周≥1次）和DHA高水平与痴呆风险降低20-40%相关。但RCT补充剂结果复杂：DHA对轻度认知障碍(MCI)有中度获益，但对已确诊阿尔茨海默病效果有限。DHA补充需长期（>2年）才能显示效果。APOE4携带者可能需要更高的DHA水平。 |
| **实践意义** | 每周2-3次深海鱼（三文鱼、沙丁鱼）是最佳来源。DHA补充剂对MCI阶段可能有益 |
| **关联器官** | 大脑与神经系统（突触膜DHA组成）、血液与血管（脑血流） |
| **关联营养元素** | Omega-3脂肪酸（DHA优先） |

## BRN-003：睡眠对大脑清除代谢废物的关键作用（类淋巴系统）

| 字段 | 内容 |
|------|------|
| **来源** | Science / Nature / 多项脑成像研究 |
| **权威等级** | 🔬 最高 |
| **证据等级** | A级（机制研究+多项人体实验） |
| **源文献** | Xie L, et al. "Sleep drives metabolite clearance from the adult brain." Science. 2013. + Fultz NE, et al. "CSF influx during NREM sleep." Science. 2019. |
| **知识内容** | 深度睡眠(NREM)期间，脑内类淋巴系统（glymphatic system）清除效率增加60%。脑脊液流入增加，清除β-淀粉样蛋白和tau蛋白等神经毒性废物。睡眠剥夺（<6小时）与Aβ沉积增加和认知下降相关。睡眠潜伏期延长（>30分钟）和睡眠片段化也是风险因素。 |
| **实践意义** | 每晚7-8小时优质睡眠是最重要的脑保护行为。优先保证NREM睡眠（前半夜） |
| **关联器官** | 大脑与神经系统（废物清除）、内分泌系统（褪黑素、皮质醇节律） |
| **关联营养元素** | 镁（助眠）、咖啡因（戒断时间） |

## BRN-004：咖啡因对认知功能的急性和慢性影响

| 字段 | 内容 |
|------|------|
| **来源** | Nature Reviews Neuroscience / 多项Meta分析 |
| **权威等级** | 🔬 最高 |
| **证据等级** | A级（多项RCT+流行病学研究） |
| **源文献** | Nehlig A. "Effects of coffee/caffeine on brain health." Pract Neurol. 2016. + 多项前瞻性队列研究综合 |
| **知识内容** | 急性：咖啡因阻断腺苷A2A受体，提升警觉度、注意力和反应时间（效应量d=0.3-0.5），持续3-6小时。慢性：长期适度咖啡摄入（3-5杯/天）与帕金森病风险降低25-30%、阿尔茨海默病风险降低20-30%相关。但>600mg/天（≈6杯）可能增加焦虑和睡眠障碍。个体CYP1A2基因型决定代谢速度：慢代谢者咖啡因半衰期可达8-10小时。 |
| **实践意义** | 早晨或下午早些时候饮用（避免影响睡眠）。3-4杯/天（300-400mg咖啡因）似乎是理想剂量 |
| **关联器官** | 大脑与神经系统（腺苷系统）、内分泌系统（皮质醇） |
| **关联营养元素** | 咖啡因 |

## BRN-005：肌酸对认知功能的增强作用

| 字段 | 内容 |
|------|------|
| **来源** | Neuroscience & Biobehavioral Reviews / 多项RCT综合 |
| **权威等级** | 🔬 高 |
| **证据等级** | B级（RCT，效果因条件而异） |
| **源文献** | Avgerinos KI, et al. "Effects of creatine supplementation on cognitive function." Neurosci Biobehav Rev. 2018. + McMorris T, et al. Psychopharmacology. 2007. |
| **知识内容** | 肌酸补充（5-20g/天）可：①改善素食者/老年人（低基线肌酸水平者）的记忆和智力表现；②在睡眠剥夺或精神疲劳时保护认知功能；③对健康年轻杂食者效果较小。脑肌酸水平约需2-4周达到稳定。机制：增加脑磷酸肌酸池→维持ATP→支持高能耗认知过程。 |
| **实践意义** | 老年人、素食者和高压脑力工作者可考虑肌酸补充（3-5g/天） |
| **关联器官** | 大脑与神经系统（能量代谢）、肌肉系统（肌酸主要储存库） |
| **关联营养元素** | 肌酸 |
