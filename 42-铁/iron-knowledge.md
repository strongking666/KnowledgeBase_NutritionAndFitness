## FE-001：运动员口服铁补充效果Meta分析

| 字段 | 内容 |
|------|------|
| **源头** | Sports Medicine |
| **权威等级** | 🥇 最高（系统综述与Meta分析，13项RCT，449人） |
| **证据等级** | A级 |
| **源文献** | "Effects of Oral Iron Supplementation on Blood Iron Status in Athletes: A Systematic Review, Meta-Analysis and Meta-Regression." Sports Med. 2024;54:1957-1976. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/38407751/ |
| **知识内容** | 口服铁补充显著提高运动员血清铁蛋白（SMD=1.27, p=0.006），但对血红蛋白的提升未达统计显著性。VO2max有积极改善趋势（SMD=0.49）。铁补充对铁储备不足的运动员最有效。 |
| **实践意义** | 运动员应在血液检测确认铁储备不足后再补充，盲目补铁效果有限 |
| **关联器官** | 血液与血管（血红蛋白）、肌肉系统（肌红蛋白/有氧能力） |
| **关联营养元素** | 铁+维生素C（增强吸收） |

## FE-002：女性运动员铁缺乏与运动表现

| 字段 | 内容 |
|------|------|
| **源头** | Journal of Sport and Health Science |
| **权威等级** | 🥇 最高（系统综述，23项研究，669名运动员） |
| **证据等级** | A级 |
| **源文献** | Pengelly M, et al. "Iron deficiency, supplementation, and sports performance in female athletes: A systematic review." J Sport Health Sci. 2025;14:101009. |
| **URL** | https://www.sciencedirect.com/science/article/pii/S2095254624001674 |
| **知识内容** | 高达60%的女性运动员存在铁缺乏（血清铁蛋白<40µg/L）。铁缺乏使耐力表现下降3-4%。补充100mg元素铁/天可改善缺铁运动员的表现。补充16-60mg/天×42-56天可提升能量效率和工作率。 |
| **实践意义** | 女性运动员应定期检测铁蛋白，铁缺乏是运动表现下降的常见原因 |
| **关联器官** | 血液与血管、肌肉系统（有氧耐力） |
| **关联营养元素** | 铁+维生素C（吸收增强）、血红素铁（动物来源吸收率更高） |

## FE-003：铁缺乏与贫血对运动表现的影响（2026 Meta）

| 字段 | 内容 |
|------|------|
| **源头** | Journal of Trace Elements and Minerals |
| **权威等级** | 🥇 最高（系统综述与Meta分析，15项研究，380人） |
| **证据等级** | A级 |
| **源文献** | Nakamoto FP, et al. "Effect of iron supplementation on exercise performance of women with NAID or IDA." J Trace Elem Miner. 2026;15:100275. |
| **知识内容** | 铁补充对缺铁女性VO2max/peak有中等程度改善（SMD=0.70, p=0.030），但对运动至力竭时间和计时赛表现改善未达显著。排除贫血者后VO2max改善也不再显著——提示贫血程度越重，补铁获益越大。 |
| **实践意义** | 缺铁性贫血者补铁收益最大；单纯缺铁者效果有限，饮食调整优先 |
| **关联器官** | 血液与血管（红细胞生成）、心脏（氧输送） |
| **关联营养元素** | 铁-血红素铁（红肉/肝脏/贝类）> 非血红素铁（菠菜/豆类） |

## FE-004：维生素C协同铁吸收

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / Clinical Nutrition ESPEN |
| **权威等级** | 🥇 最高（系统综述与Meta分析） |
| **证据等级** | A级 |
| **源文献** | Loganathan V, et al. "Treatment efficacy of vitamin C or ascorbate given as co-intervention with iron for anemia." Clin Nutr ESPEN. 2023;57:459-468. |
| **知识内容** | 维生素C与铁同时服用可显著增强铁吸收（尤其非血红素铁）。维生素C将Fe³⁺还原为更易吸收的Fe²⁺，并形成可溶性螯合物。50-100mg维生素C可使非血红素铁吸收率提高2-3倍。 |
| **实践意义** | 吃含铁食物时搭配含维生素C的食物（如菠菜+柠檬汁、牛肉+青椒） |
| **关联器官** | 消化系统（肠道吸收）、血液与血管 |
| **关联营养元素** | 铁+维生素C（协同吸收） |
