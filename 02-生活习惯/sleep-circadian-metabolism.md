## SLE-001：社交时差与肥胖风险

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / Obesity Reviews |
| **权威等级** | 🥇 最高（系统综述与Meta分析） |
| **证据等级** | A级 |
| **源文献** | "Social jetlag and obesity: A systematic review and meta-analysis." Obes Rev. 2024. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/38072635/ |
| **知识内容** | 社交时差（工作日与周末作息差异≥1小时）与超重/肥胖显著相关。每增加1小时社交时差，肥胖风险上升。机制：昼夜节律紊乱→食欲激素失调（瘦素↓、饥饿素↑）→代谢紊乱。 |
| **实践意义** | 保持工作日和周末一致的睡眠时间，减少社交时差 |
| **关联器官** | 大脑与神经系统（昼夜节律中枢）、内分泌系统（激素调节） |
| **关联营养元素** | 昼夜节律-进食时机 |

## SLE-002：限时进食与昼夜节律（时间营养学）

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / Nutrients |
| **权威等级** | 🥇 最高（系统综述） |
| **证据等级** | A级 |
| **源文献** | "Food Timing, Circadian Rhythm and Chrononutrition: A Systematic Review of Time-Restricted Eating's Effects on Human Health." Nutrients. 2020;12(12):3770. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/33302500/ |
| **知识内容** | 进食时间与昼夜节律同步对代谢健康至关重要。晚间进食（与生物钟错位）与肥胖、胰岛素抵抗、心血管疾病风险增加相关。TRE（限时进食）通过将进食窗口限制在白天活跃期，改善血糖控制、胰岛素敏感性和体重管理。 |
| **实践意义** | 尽量在白天（6:00-18:00）完成主要进食，避免深夜进食 |
| **关联器官** | 内分泌系统（胰岛素、褪黑素）、消化系统（胃肠道节律） |
| **关联营养元素** | 进食时机-时间营养学 |

## SLE-003：睡眠剥夺对心脏自主神经的影响

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / Frontiers in Neurology |
| **权威等级** | 🥇 最高（系统综述与Meta分析，11项RCT） |
| **证据等级** | A级 |
| **源文献** | "Effects of sleep deprivation on heart rate variability: a systematic review and meta-analysis." Front Neurol. 2025. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/40895095/ |
| **知识内容** | 睡眠剥夺导致心率变异性（HRV）显著降低（RMSSD↓），交感神经活性增加（LF↑、LF/HF↑），副交感（迷走）神经活性抑制。表明睡眠不足使心脏自主神经平衡偏向交感优势，增加心血管负担。 |
| **实践意义** | 每晚7-9小时充足睡眠对心脏自主神经健康至关重要 |
| **关联器官** | 心脏（HRV、自主神经调节）、大脑与神经系统（自主神经中枢） |
| **关联营养元素** | 镁（促进睡眠）、褪黑素 |

## SLE-004：夜间光照与肥胖风险

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / BMC Public Health |
| **权威等级** | 🥇 最高（系统综述与Meta分析，13项研究，86万人） |
| **证据等级** | A级 |
| **源文献** | "Impact of artificial light at night on obesity and overweight: a systematic review and meta-analysis." BMC Public Health. 2025. |
| **URL** | https://pmc.ncbi.nlm.nih.gov/articles/PMC12817692/ |
| **知识内容** | 夜间人造光照暴露与肥胖/超重风险增加显著相关。机制：光照抑制褪黑素分泌→昼夜节律紊乱→代谢失调。亚洲人群的关联比欧美更强。 |
| **实践意义** | 睡前1-2小时减少屏幕蓝光暴露，使用暖色灯光或蓝光过滤 |
| **关联器官** | 大脑与神经系统（松果体/褪黑素）、内分泌系统（代谢） |
| **关联营养元素** | 褪黑素（天然来源：酸樱桃、核桃） |

## 知识源交叉验证

✅ 社交时差 + 夜间光照 + 睡眠剥夺 → 一致指向昼夜节律紊乱的代谢危害
✅ TRE（时间营养学）→ 进食-生物钟同步是有效的代谢健康策略
✅ 睡眠不足→ 心脏自主神经失衡（交感↑、迷走↓）
