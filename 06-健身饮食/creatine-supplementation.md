## CRE-001：肌酸补充的安全性与功效（ISSN立场声明）

| 字段 | 内容 |
|------|------|
| **源头** | ISSN（国际运动营养学会） |
| **权威等级** | 🥇 最高（专业学会立场声明） |
| **证据等级** | A级（数百项RCT基础） |
| **源文献** | Kreider RB, et al. "International Society of Sports Nutrition position stand: safety and efficacy of creatine supplementation in exercise, sport, and medicine." J Int Soc Sports Nutr. 2017;14:18. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/28615996/ |
| **知识内容** | 肌酸是研究最充分、最有效的运动营养补剂之一。补充可增加肌肉肌酸和磷酸肌酸水平20-40%。长期补充（高达30g/天×5年）在健康人群和各年龄段患者中均安全且耐受良好。唯一明确副作用是体重增加（主要是水潴留）。对肾脏功能无不良影响（健康人群）。 |
| **实践意义** | 肌酸一水化物是最有效且性价比最高的补剂形式 |
| **关联器官** | 肌肉系统（增力/增肌）、大脑与神经系统（神经保护）、心脏（缺血保护） |
| **关联营养元素** | 肌酸-一水化物(CrM) |

## CRE-003：肌酸补充方案

| 字段 | 内容 |
|------|------|
| **源头** | ISSN立场声明 |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | Kreider RB, et al. JISSN. 2017 |
| **知识内容** | **加载期方案**：0.3g/kg/天（约20-25g/天）分4次×5-7天，然后3-5g/天维持。**慢速方案**：3-5g/天×3-4周达到同样饱和。与碳水/蛋白质共摄入可增加肌酸保留。**素食者**基线肌酸水平低，补充效果更明显。 |
| **实践意义** | 推荐3-5g/天维持方案（简单高效）；加碳水可增加吸收 |
| **关联器官** | 肌肉系统（肌酸储留） |
| **关联营养元素** | 肌酸+碳水化合物（共摄入增强吸收） |

## CRE-004：肌酸的临床应用

| 字段 | 内容 |
|------|------|
| **源头** | ISSN立场声明 / 多项RCT |
| **权威等级** | 🥈 高 |
| **证据等级** | B-C级（部分领域RCT较少） |
| **知识内容** | 肌酸在以下临床领域有潜在益处：①肌少症（老年人保肌）；②2型糖尿病（改善血糖控制+GLUT-4转位）；③神经退行性疾病（帕金森、亨廷顿）；④脑外伤/脊髓损伤（神经保护）；⑤心脏缺血保护；⑥孕期/青少年抑郁辅助。大多数应用仍需要更多RCT证据。 |
| **实践意义** | 老年人补充肌酸可能有助于延缓肌少症 |
| **关联器官** | 肌肉系统、大脑与神经系统、心脏、内分泌系统 |
| **关联营养元素** | 肌酸 |
