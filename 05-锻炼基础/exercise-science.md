## EXS-001：超量恢复理论——从糖原到线粒体的训练适应

| 字段 | 内容 |
|------|------|
| **源头** | Sports Medicine / Journal of Applied Physiology |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | Hawley JA, et al. "Training techniques to improve fatigue resistance and enhance endurance performance." J Sports Sci. 1997;15(3):325-333. + Booth FW, Thomason DB. "Molecular and cellular adaptation of muscle in response to exercise: perspectives of various models." Physiol Rev. 1991;71(2):541-585. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/9232558/ |
| **知识内容** | 超量恢复(Supercompensation)是训练适应的基础理论：①训练刺激→暂时性功能下降（糖原耗竭、力量降低）→恢复期→超越原有水平（超量恢复）→若未及时再刺激则回落至基线。不同系统恢复时间不同：肌糖原24-48h（高碳水饮食可缩短至24h）、肌肉力量48-72h（DOMS后）、线粒体酶活性数天至数周。关键应用：①训练频率需匹配目标系统的恢复节奏；②"训练-恢复-适应"的闭环中，恢复（营养+睡眠）与训练同等重要；③慢性超量恢复积累→长期训练适应（线粒体密度↑、糖原储备↑、力量↑）。 |
| **实践意义** | 训练收益来自恢复期的超量恢复，恢复（营养+睡眠）与训练同等重要 |
| **关联器官** | 肌肉系统（糖原、蛋白合成）、线粒体（密度与功能） |
| **关联营养元素** | 碳水化合物（糖原再合成）、蛋白质（肌蛋白修复） |

## EXS-002：RPE（自感用力评分）的科学基础——主观感受的生理锚定

| 字段 | 内容 |
|------|------|
| **源头** | Medicine & Science in Sports & Exercise / Sports Medicine |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | Borg GA. "Psychophysical bases of perceived exertion." Med Sci Sports Exerc. 1982;14(5):377-381. + Eston R. "Use of ratings of perceived exertion in sports." Int J Sports Physiol Perform. 2012;7(2):175-182. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/7154893/ |
| **知识内容** | RPE（Borg 6-20量表）的生理学锚定：①6-20对应静息心率(60 bpm)到最大心率(200 bpm)（RPE×10≈心率），非巧合；②RPE与多个生理参数高度相关——心率(r=0.80-0.90)、血乳酸(r=0.80-0.87)、VO2(r=0.85)、通气量(r=0.85)；③CR-10量表（0-10）更适合评估疼痛/局部肌肉疲劳；④RPE可靠性——重测信度高（ICC>0.85），训练者比未训练者更准确；⑤RPE可替代生理测试进行训练强度监控（尤其缺乏心率监测设备时）。关键：RPE不仅是"主观感受"，而是整合了中枢和周围信号的可靠生理反馈。 |
| **实践意义** | RPE是经过生理锚定的可靠训练强度工具，可替代HR监控 |
| **关联器官** | 中枢神经系统（主观感知整合）、心血管系统（HR对应） |
| **关联营养元素** | 血糖（低血糖→RPE↑） |

## EXS-003：心率变异性(HRV)指导训练——比固定计划更智能？

| 字段 | 内容 |
|------|------|
| **源头** | International Journal of Sports Physiology and Performance / Frontiers in Physiology |
| **权威等级** | 🥈 高 |
| **证据等级** | B级（RCT证据有前景但非完全一致） |
| **源文献** | Kiviniemi AM, et al. "Endurance training guided individually by daily heart rate variability measurements." Eur J Appl Physiol. 2007;101(6):743-751. + Javaloyes A, et al. "Training prescription guided by heart rate variability in cycling." Int J Sports Med. 2019;40(3):181-187. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/17724625/ |
| **知识内容** | HRV（逐次心跳间期的微小变化）反映自主神经系统平衡——高HRV（副交感占优）=恢复良好，低HRV（交感占优）=疲劳/压力。HRV指导训练的经典RCT(Kiviniemi 2007)：HRV指导组（低HRV日降强度或休息，高HRV日高强度）vs固定计划组——8周后HRV组VO2max提升（+8.2% vs +4.5%，p<0.05），训练量更合理分布。但2024年系统综述结论更谨慎：HRV指导训练可能优于固定计划（效应量0.3-0.4），但优势主要体现在避免过度训练和优化恢复，而非必然提升峰值表现。实用建议：HRV作为辅助决策工具而非唯一依据。 |
| **实践意义** | HRV指导训练可优化恢复和避免过度训练，但不必然提升峰值表现 |
| **关联器官** | 心脏（自主神经调节）、中枢神经系统 |
| **关联营养元素** | 镁（可能影响自主神经张力） |

## EXS-004：VO2max与寿命——心肺适能作为全因死亡率的最强预测因子

| 字段 | 内容 |
|------|------|
| **源头** | JAMA / JAMA Network Open / Progress in Cardiovascular Diseases |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级（大规模前瞻性队列+Meta分析） |
| **源文献** | Mandsager K, et al. "Association of cardiorespiratory fitness with long-term mortality among adults undergoing exercise treadmill testing." JAMA Netw Open. 2018;1(6):e183605. + Kodama S, et al. "Cardiorespiratory fitness as a quantitative predictor of all-cause mortality and cardiovascular events in healthy men and women: a meta-analysis." JAMA. 2009;301(19):2024-2035. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/30646252/ |
| **知识内容** | VO2max（最大摄氧量）是全因死亡率的最强独立预测因子之一。Mandsager(2018)：122,007人随访8.4年→高心肺适能组（>90th百分位）全因死亡率比低适能组（<25th百分位）低约80%（HR 0.20），远强于吸烟、高血压、糖尿病的预测力。Kodama(2009)Meta分析（33项研究，187,303人）：每增加1 MET（≈3.5 mL/kg/min）心肺适能→全因死亡率降低13%，心血管死亡率降低15%。"最低适能"到"低适能"的跃迁获益最大（大部分人口可通过每周150分钟中等强度运动实现）。"坐着不动"比吸烟更致命。 |
| **实践意义** | 提高VO2max是延长寿命最有效的非药物干预 |
| **关联器官** | 心脏、肺、血管系统、线粒体 |
| **关联营养元素** | 硝酸盐（蔬菜→NO→VO2微幅提升）、铁（血红蛋白合成） |
