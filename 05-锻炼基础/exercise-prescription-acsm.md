## EX-002：ACSM运动测试与处方指南第12版（2025）

|------|------|

| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | ACSM's Guidelines for Exercise Testing and Prescription, 12th Ed. Wolters Kluwer, 2025. |
| **知识内容** | **FITT-VP原则**：Frequency(频率)、Intensity(强度)、Time(时间)、Type(类型)、Volume(总量)、Progression(进阶)。**综合运动计划**应包括有氧+抗阻+柔韧性+神经运动训练。目标能量消耗：500-1000 MET-分钟/周。**进阶**：初始阶段每1-2周增加5-10分钟，之后调整强度或频率。新增章节：影响大脑的疾病（帕金森、阿尔茨海默、抑郁、焦虑）。 |

| **关联器官** | 心脏（有氧）、肌肉（抗阻）、大脑与神经系统（神经运动）、骨骼与关节（柔韧） |
| **关联营养元素** | 碳水化合物（运动供能）、蛋白质（恢复） |

## EX-003：VO2max与健康寿命

| 字段 | 内容 |
|------|------|
| **源头** | ACSM / 多项大型队列研究 |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | Multiple large cohort studies + ACSM guidelines |
| **知识内容** | VO2max（最大摄氧量）是心肺适能的黄金标准，也是全因死亡率的强预测因子。每增加1 MET（约3.5 ml/kg/min）的VO2max，全因死亡率降低约10-15%。低VO2max比吸烟、高血压、高胆固醇等传统风险因素更强的死亡预测力。VO2max可通过任何形式的有氧训练提升8-20%。 |
| **实践意义** | 提升VO2max是延长健康寿命最有效的生活方式干预之一 |
| **关联器官** | 心脏（心输出量）、肺脏（氧摄取）、血液与血管（氧运输）、肌肉系统（氧利用） |
| **关联营养元素** | 铁（血红蛋白/氧运输） |

## EX-004：运动恢复与过度训练

| 字段 | 内容 |
|------|------|
| **源头** | ACSM / NSCA / 多项系统综述 |
| **权威等级** | 🥈 高 |
| **证据等级** | B级 |
| **源文献** | ACSM Guidelines + multiple systematic reviews |
| **知识内容** | **过度训练综合征**：表现为持续疲劳、运动表现下降、情绪紊乱、睡眠障碍。预防：①周期性减量（每4-6周减量1周）；②充足的睡眠（>7小时）；③合理的营养（足够热量+蛋白质）；④主动恢复（低强度运动促进血流量和代谢废物清除）。**恢复指标**：静息心率升高>5bpm、HRV降低、主观疲劳感增加。 |
| **实践意义** | 训练进步发生在恢复期而非训练期——安排充足的恢复 |
| **关联器官** | 肌肉系统、内分泌系统（皮质醇/睾酮比值）、大脑与神经系统（中枢疲劳） |
| **关联营养元素** | 蛋白质、碳水化合物（糖原恢复）、镁（肌肉放松） |
