## WL-001：热量缺口与减重（500kcal法则）

| 字段 | 内容 |
|------|------|
| **源头** | Dietary Guidelines for Americans / PubMed |
| **权威等级** | 🥇 最高（政府膳食指南+RCT验证） |
| **证据等级** | A级 |
| **源文献** | Carels RA, et al. "Can following the caloric restriction recommendations from the Dietary Guidelines for Americans help individuals lose weight?" Eat Behav. 2008; (54名超重/肥胖成人，14周) |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/18549992/ |
| **知识内容** | 每日创造≥500kcal热量缺口的个体，14周减重效果是缺口不足500kcal者的近4倍（p<0.01）。成功减重5%体重的个体自我监测天数是未达标者的2倍以上。 |
| **实践意义** | 每日500kcal缺口 + 规律自我监测（记录饮食/运动）= 有效减重核心策略 |
| **关联器官** | 全身（脂肪组织）、内分泌系统（瘦素/饥饿素调节） |
| **关联营养元素** | 能量平衡-热量缺口 |

## WL-002：间歇性vs持续性热量限制

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / Journal of Translational Medicine |
| **权威等级** | 🥇 最高（系统综述与Meta分析） |
| **证据等级** | A级（11项RCT） |
| **源文献** | "Intermittent versus continuous energy restriction on weight loss and cardiometabolic outcomes." J Transl Med. 2018;16(1):371. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/30583725/ |
| **知识内容** | 间歇性热量限制（如5:2饮食）与持续性热量限制在减重效果上无显著差异（WMD: -0.61kg, p=0.87）。间歇性方案对空腹胰岛素有微弱优势。两者均有效，选择取决于个人偏好和依从性。 |
| **实践意义** | 选择你最能坚持的方案——间歇性或持续性都可以有效减重 |
| **关联器官** | 内分泌系统（胰岛素敏感性）、全身（脂肪） |
| **关联营养元素** | 能量平衡-热量限制 |

## WL-003：限时进食(TRE)减重效果（2026最新Meta）

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / Nutrition Reviews |
| **权威等级** | 🥇 最高（系统综述与Meta分析） |
| **证据等级** | A级（RCT综合） |
| **源文献** | Fernandes-Alves D, et al. "Systematic Review and Meta-analysis of RCTs Comparing Time-Restricted Eating With and Without Caloric Restriction for Weight Loss." Nutr Rev. 2026;84(3):463-486. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/40298934/ |
| **知识内容** | TRE（限时进食，通常8-10小时进食窗口）即使不刻意计算热量也能产生减重效果，但配合热量限制效果更好。TRE的减重主要通过自然减少进食机会和改善昼夜节律实现，而非代谢魔法。 |
| **实践意义** | 8-10小时进食窗口+不暴食=有效且简单的减重策略 |
| **关联器官** | 内分泌系统（昼夜节律、胰岛素）、消化系统 |
| **关联营养元素** | 进食时机-限时进食 |

## WL-004：减重后宏量营养素对静息能量消耗的影响（2024 Meta）

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / Obesity Reviews |
| **权威等级** | 🥇 最高（系统综述与Meta分析） |
| **证据等级** | A级 |
| **源文献** | Ho DKN, et al. "The effects of dietary macronutrient composition on resting energy expenditure following active weight loss." Obes Rev. 2024;25(8):e13760. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/38697953/ |
| **知识内容** | 减重后不同宏量营养素比例影响静息能量消耗（REE）。较高蛋白质+较低血糖指数的饮食模式有助于维持较高的REE，减少减重后的代谢适应（代谢减慢），有利于体重维持。 |
| **实践意义** | 减重后保持较高蛋白质摄入有助于防止反弹 |
| **关联器官** | 肌肉系统（维持瘦体重）、内分泌系统（代谢调节） |
| **关联营养元素** | 蛋白质（维持REE）、碳水化合物-GI指数 |

## 知识源交叉验证

✅ Dietary Guidelines + 多篇Meta分析 → 热量缺口是减重的必要条件
✅ 间歇性vs持续性 → 效果相当，依从性是关键
✅ 减重后高蛋白饮食 → 有助于减少代谢适应和体重反弹
