# 营养管理与健身知识库

> **Nutrition & Fitness Knowledge Base**
> 
> 本知识库为AI产品提供权威的营养管理与健身知识依据。
> 知识来源以国际权威文献、官方机构指南为准。

## 知识源优先级

1. 🥇 **国际权威文献** - PubMed, Cochrane, WHO, NIH
2. 🥈 **官方机构指南** - ACSM, ISSN, AND, WHO指南
3. 🥉 **权威教材** - 运动生理学、营养学经典教材
4. 📊 **Meta分析/系统综述** - 高证据等级研究

## 数据格式

每条知识记录包含：
- 源头标签 (Source)
- 权威等级 (Authority Level)
- 类别标签 (Category)
- 源文献名 (Reference)
- 知识内容 (Content)
- 最后更新 (Last Updated)
