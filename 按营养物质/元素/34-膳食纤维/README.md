# 膳食纤维 (膳食纤维)

> **维度**: 营养物质/元素
> **关联知识源标签**: 营养-膳食纤维

## 常见来源及推荐排序

按补充效果从高到低排序：

1. 燕麦麸
2. 洋车前子壳
3. 豆类(黑豆/鹰嘴豆)
4. 奇亚籽
5. 西兰花
6. 苹果(带皮)
7. 全谷物
8. 坚果

## 缺乏或过量影响

- 缺乏: 便秘、憩室病、结直肠癌风险增加、胆固醇升高、血糖波动
- 过量: 腹胀、胀气、矿物质吸收障碍、肠梗阻(极端)

## 推荐家常菜系及做法

- 燕麦粥配浆果
- 鹰嘴豆沙拉
- 奇亚籽布丁
- 西兰花炒虾仁
- 全麦面包
- 蒸豆角
