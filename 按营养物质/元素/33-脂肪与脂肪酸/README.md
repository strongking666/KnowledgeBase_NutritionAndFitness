# 脂肪与脂肪酸 (脂肪与脂肪酸)

> **维度**: 营养物质/元素
> **关联知识源标签**: 营养-脂肪与脂肪酸

## 常见来源及推荐排序

按补充效果从高到低排序：

1. 橄榄油(单不饱和)
2. 牛油果
3. 坚果(核桃/杏仁)
4. 三文鱼(Omega-3)
5. 亚麻籽油
6. 椰子油(MCT)
7. 蛋黄

## 缺乏或过量影响

- 缺乏: 必需脂肪酸缺乏、激素合成障碍、脂溶性维生素吸收不良
- 过量: 肥胖、心血管疾病、脂肪肝、炎症增加(饱和脂肪过量)

## 推荐家常菜系及做法

- 橄榄油拌沙拉
- 牛油果吐司
- 三文鱼刺身
- 亚麻籽酸奶
- 坚果混合零食
