## VITA-001：维生素A在视觉周期中的分子机制（视紫红质再生）

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / Annual Review of Nutrition / 经典分子生物学教材 |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级（分子机制已完全阐明） |
| **源文献** | Wald G. "The Molecular Basis of Visual Excitation." Nobel Lecture, 1967. + Saari JC. "Vitamin A Metabolism in Rod and Cone Visual Cycles." Annu Rev Nutr. 2012;32:125-145. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/22809103/ |
| **知识内容** | 维生素A在视觉周期中的核心角色：①11-顺式视黄醛与视蛋白结合形成视紫红质（rhodopsin），位于视杆细胞外段膜盘；②光子使11-顺式转变为全反式视黄醛，触发G蛋白偶联级联反应→cGMP水解→Na+通道关闭→超极化→神经信号；③全反式视黄醛需经RPE65酶重新异构化回11-顺式，完成视循环再生；④一个光子可激活一个视紫红质分子，放大信号约10^5倍。维生素A缺乏首先表现为暗适应延长（夜盲），因视紫红质再生受阻。 |
| **实践意义** | 夜盲是维生素A缺乏最早的特异性症状，可作为早期筛查指标 |
| **关联器官** | 视觉系统（视网膜-视杆细胞/视锥细胞） |
| **关联营养元素** | 维生素A-视黄醛-视紫红质、锌（RBP合成辅因子） |

## VITA-002：β-胡萝卜素转化为视黄醇的效率差异（基因多态性影响）

| 字段 | 内容 |
|------|------|
| **源头** | American Journal of Clinical Nutrition / FASEB Journal |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | Lietz G, et al. "Single nucleotide polymorphisms upstream from the β-carotene 15,15'-monoxygenase gene influence provitamin A conversion efficiency in female volunteers." J Nutr. 2009;139(4):654-659. + Hickenbottom SJ, et al. "Variability in conversion of β-carotene to vitamin A in men." Am J Clin Nutr. 2002;75(5):900-907. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/19244379/ |
| **知识内容** | β-胡萝卜素向维生素A（视黄醇）的转化效率存在巨大个体差异：①传统认为6μg膳食β-胡萝卜素=1μg视黄醇当量（RE），后修正为12:1（RDA），近年发现实际为21:1-28:1；②β-胡萝卜素15,15'-单加氧酶1（BCMO1）基因SNP（rs12934922、rs7501331）使转化效率降低30-70%；③女性转化效率平均高于男性；④肠道寄生虫、脂肪吸收不良、甲状腺功能减退进一步降低转化率；⑤纯素食者完全依赖植物来源时，BCMO1低活性基因型风险较高。 |
| **实践意义** | 纯素食者和BCMO1低活性基因型者应优先考虑直接摄入动物来源视黄醇或强化食品 |
| **关联器官** | 消化系统（肠道-吸收）、肝脏（储存与代谢） |
| **关联营养元素** | 维生素A-β-胡萝卜素、膳食脂肪（促吸收）、锌 |

## VITA-003：维生素A对免疫功能的支持（尤其是黏膜免疫和麻疹死亡率）

| 字段 | 内容 |
|------|------|
| **源头** | WHO / Lancet / BMJ / 多项系统综述 |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | WHO. "Vitamin A supplementation in infants and children 6-59 months of age." WHO Guideline, 2011. + Sommer A, et al. "Impact of vitamin A supplementation on childhood mortality." Lancet. 1986;1:1169-1173. + Mayo-Wilson E, et al. "Vitamin A supplements for preventing mortality, illness, and blindness in children aged under 5." BMJ. 2011;343:d5094. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/21868478/ |
| **知识内容** | 维生素A被称为"抗感染维生素"：①维持黏膜上皮完整性（呼吸道、胃肠道、泌尿道第一道防线），缺乏导致鳞状上皮化生→病原体入侵门户；②调节先天免疫：促进NK细胞活性和巨噬细胞吞噬；③支持适应性免疫：促进T细胞增殖分化和B细胞抗体产生（尤其IgA）；④大样本RCT Meta分析（17项研究，194,000+儿童）：维生素A补充使全因死亡率降低24%，麻疹死亡率降低50%；⑤WHO推荐6-59月龄儿童每4-6个月补充维生素A（高剂量，100,000-200,000 IU）。 |
| **实践意义** | 维生素A缺乏地区儿童补充是WHO最成本有效的公共卫生干预之一 |
| **关联器官** | 免疫系统（黏膜免疫、淋巴细胞）、呼吸系统、消化系统 |
| **关联营养元素** | 维生素A-免疫功能、锌（协同免疫支持） |

## VITA-004：维生素A过量的肝毒性——安全上限与北极探险者案例

| 字段 | 内容 |
|------|------|
| **源头** | NEJM / 医学史文献 / EFSA / 中国营养学会 |
| **权威等级** | 🥇 最高 |
| **证据等级** | B级（案例报道+动物实验+观察性研究） |
| **源文献** | Rodahl K, Moore T. "The vitamin A content and toxicity of bear and seal liver." Biochem J. 1943;37(2):166-168. + Geubel AP, et al. "Liver damage caused by therapeutic vitamin A administration." Gastroenterology. 1991;100(6):1701-1709. + EFSA. "Tolerable Upper Intake Levels for Vitamins and Minerals." 2006. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/1674768/ |
| **知识内容** | 维生素A是少数容易过量的脂溶性维生素：①北极探险者食用北极熊肝脏后出现急性维生素A中毒（嗜睡、头痛、呕吐、皮肤脱屑），因北极熊肝维生素A含量高达18,000 IU/g（安全上限的450倍）；②慢性维生素A中毒（>10,000 IU/天，数月至数年）：肝星状细胞储存饱和→肝窦纤维化→非酒精性脂肪肝样病变→门静脉高压；③中国居民膳食营养素参考摄入量：UL=3,000 μg RAE/天（约10,000 IU）；④孕期维生素A过量致畸（>10,000 IU/天），尤其是孕早期神经嵴发育期；⑤异维A酸（维生素A衍生物）明确致畸，育龄女性使用期间需严格避孕。 |
| **实践意义** | 日常饮食+复合维生素一般安全；避免同时服用多种含维生素A的补充剂；孕期避免高剂量 |
| **关联器官** | 肝脏（肝星状细胞-纤维化）、神经系统（急性中毒-假性脑瘤）、胚胎（致畸） |
| **关联营养元素** | 维生素A-视黄醇棕榈酸酯（肝储存形式） |
