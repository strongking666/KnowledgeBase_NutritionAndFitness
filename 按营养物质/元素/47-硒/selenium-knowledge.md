## SEL-001：硒蛋白与抗氧化——谷胱甘肽过氧化物酶的硒依赖

| 字段 | 内容 |
|------|------|
| **源头** | Annual Review of Nutrition / Free Radical Biology and Medicine / Biochimica et Biophysica Acta |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级（分子机制完全阐明） |
| **源文献** | Rotruck JT, et al. "Selenium: biochemical role as a component of glutathione peroxidase." Science. 1973;179(4073):588-590. + Brigelius-Flohé R, Maiorino M. "Glutathione peroxidases." Biochim Biophys Acta. 2013;1830(5):3289-3303. + Labunskyy VM, et al. "Selenoproteins: molecular pathways and physiological roles." Physiol Rev. 2014;94(3):739-777. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/22967787/ |
| **知识内容** | 硒的抗氧化功能完全依赖其作为硒蛋白的组成成分：①硒以硒代半胱氨酸（Sec，第21种氨基酸）形式共翻译插入硒蛋白——UGA密码子在SECIS元件存在时编码Sec而非终止；②谷胱甘肽过氧化物酶家族（GPX1-8）：GPX1（胞质）和GPX4（膜/磷脂氢过氧化物）催化GSH+H₂O₂/ROOH→GSSG+H₂O/ROH，硒代半胱氨酸位于活性中心；③硫氧还蛋白还原酶（TrxR1-3）：含硒蛋白，还原氧化硫氧还蛋白→维持细胞内还原环境→调控转录因子（NF-κB, AP-1）氧化还原状态；④GPX4是铁死亡（ferroptosis）的关键抑制因子——GPX4失活→膜磷脂过氧化物积累→铁依赖的非凋亡细胞死亡；⑤硒蛋白合成遵循层级调控——硒不足时，机体优先维持"必需"硒蛋白（GPX4, TrxR）而牺牲"非必需"（GPX1, GPX3）——血清GPX活性是硒状态的敏感指标。 |
| **实践意义** | 硒的抗氧化功能不可被其他营养素替代；硒状态不足→GPX活性下降→氧化应激易感性增加 |
| **关联器官** | 全身（细胞内抗氧化-谷胱甘肽系统）、细胞膜（GPX4-铁死亡保护） |
| **关联营养元素** | 硒-硒代半胱氨酸、谷胱甘肽（GPX底物）、维生素E（协同膜保护）、铁（铁死亡通路） |

## SEL-002：硒对甲状腺激素转化的关键作用

| 字段 | 内容 |
|------|------|
| **源头** | Endocrine Reviews / Thyroid / Lancet Diabetes & Endocrinology |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级（分子机制完全阐明） |
| **源文献** | Köhrle J, et al. "Selenium, the thyroid, and the endocrine system." Endocr Rev. 2005;26(7):944-984. + Beckett GJ, Arthur JR. "Selenium and endocrine systems." J Endocrinol. 2005;184(3):455-465. + Duntas LH, Benvenga S. "Selenium: an element for life." Endocrine. 2015;48(3):756-775. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/25870049/ |
| **知识内容** | 硒在甲状腺激素代谢中的不可替代角色：①碘甲状腺原氨酸脱碘酶（DIO1, DIO2, DIO3）均为含硒蛋白——DIO1和DIO2催化T4→T3（活性甲状腺激素）的外环脱碘，DIO3催化T4→rT3和T3→T2的内环脱碘（灭活）；②甲状腺是人体硒浓度最高的器官（每克组织含硒最高），即使在全身硒缺乏时也优先维持甲状腺硒供应；③GPX在甲状腺滤泡细胞中清除H₂O₂（甲状腺过氧化物酶合成T4时的副产物）→保护甲状腺细胞免受氧化损伤；④硒缺乏的临床后果：T4→T3转化受阻→外周组织T3水平降低、T4和rT3升高→功能性甲减；自身免疫性甲状腺炎（桥本）患者硒状态与抗体滴度负相关；⑤硒补充对自身免疫性甲状腺炎（AITD）的RCT证据：补充硒（200μg/天硒代蛋氨酸，6-12个月）可降低TPOAb滴度20-60%，改善甲状腺超声回声；⑥中国克山病地区同时存在硒缺乏和碘缺乏→甲状腺肿和甲减高发。 |
| **实践意义** | 甲状腺功能评估时应考虑硒状态；自身免疫性甲状腺炎患者可考虑硒补充（200μg/天） |
| **关联器官** | 甲状腺（滤泡细胞-T4/T3转化-抗氧化保护）、内分泌系统（下丘脑-垂体-甲状腺轴） |
| **关联营养元素** | 硒-硒代半胱氨酸（脱碘酶）、碘（协同甲状腺激素合成）、铁（TPO辅因子）、锌 |

## SEL-003：硒摄入的U型曲线——从克山病到硒中毒

| 字段 | 内容 |
|------|------|
| **源头** | Lancet / American Journal of Clinical Nutrition / 中国医学科学院 |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级（流行病学+干预试验+案例报道） |
| **源文献** | Chen X, et al. "Studies on the relations of selenium and Keshan disease." Biol Trace Elem Res. 1980;2(2):91-107. + Rayman MP. "Selenium and human health." Lancet. 2012;379(9822):1256-1268. + Yang GQ, et al. "Endemic selenium intoxication of humans in China." Am J Clin Nutr. 1983;37(5):872-881. + Vinceti M, et al. "Selenium for preventing cancer." Cochrane Database Syst Rev. 2018;1:CD005195. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/22381456/ |
| **知识内容** | 硒的摄入存在严格的U型风险曲线——不足和过量均有危害：①硒缺乏（<20μg/天）：克山病（扩张型心肌病，儿童和育龄女性高发）——中国黑龙江克山县1970s爆发，亚硒酸钠补充完全预防；大骨节病（骨关节炎+骨骺板发育异常）；②硒足量范围：约55-400μg/天——GPX和脱碘酶活性达平台，免疫功能最佳；③硒过量（>400μg/天，UL=400μg/天）：慢性硒中毒（硒质沉着病）——大蒜味口气（二甲基硒呼出）、脱发、指甲脆裂脱落、皮肤病变、周围神经病变；④1961-1964年中国湖北恩施爆发地方性硒中毒——当地土壤硒含量极高（煤矸石风化），居民日均摄入>5mg（UL的12倍），脱发率100%、指甲脱落率50%；⑤SELECT试验提示即使<UL的长期补充（200μg/天硒代蛋氨酸）也可能增加2型糖尿病风险（非显著趋势但需关注）；⑥关键教训：硒是最窄治疗窗口的必需微量元素之一——缺乏有害，过量同样有害，盲目补充不可取。 |
| **实践意义** | 不建议常规高剂量硒补充；中国从克山病区到恩施高硒区跨度极大，需根据地域和饮食评估；血清硒80-120μg/L为理想范围 |
| **关联器官** | 心脏（克山病-心肌）、骨骼与关节（大骨节病）、皮肤与毛发（硒中毒-角蛋白）、神经系统 |
| **关联营养元素** | 硒-硒代蛋氨酸/亚硒酸钠、碘（协同甲状腺）、维生素E（协同抗氧化保护） |

## SEL-004：硒对男性生育能力的营养支持

| 字段 | 内容 |
|------|------|
| **源头** | Human Reproduction / British Journal of Nutrition / Cochrane Database |
| **权威等级** | 🥈 较高 |
| **证据等级** | B级（RCT证据中等，机制明确） |
| **源文献** | Scott R, et al. "The effect of oral selenium supplementation on human sperm motility." Br J Urol. 1998;82(1):76-80. + Safarinejad MR, Safarinejad S. "Efficacy of selenium and/or N-acetyl-cysteine for improving semen parameters in infertile men." J Urol. 2009;181(2):741-751. + Ahsan U, et al. "Role of selenium in male reproduction." Anim Reprod Sci. 2014;146(1-2):55-62. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/19152964/ |
| **知识内容** | 硒在男性生育中的关键作用：①GPX4是精子结构蛋白——在精子发生后期，GPX4通过交联形成线粒体鞘（精子中段螺旋鞘），非酶结构功能——这是硒蛋白不依赖其酶活性的罕见例子；②磷脂氢过氧化物GPX（GPX4/PHGPX）保护精子膜免受脂质过氧化——精子富含多不饱和脂肪酸（膜流动性需要）但对氧化损伤高度敏感；③硒蛋白P（SEPP1）将硒从肝脏转运至睾丸——SEPP1敲除小鼠不育，补充硒可恢复；④Scott等RCT：硒补充（100μg/天，3个月）提高精子活力（+22%）和形态正常率；⑤Safarinejad等RCT（468名不育男性）：硒（200μg/天）+NAC（600mg/天）26周，精子浓度、活力、形态均显著改善，妊娠率提高（10.8% vs 安慰剂0%）；⑥但硒过量同样损害生育——高硒导致精子DNA碎片化增加、活力下降——再次体现U型曲线。 |
| **实践意义** | 男性不育检查应评估硒状态；硒充足者不必补充（U型风险）；硒缺乏地区补充可改善精液参数 |
| **关联器官** | 生殖系统（睾丸-精子发生-附睾成熟）、内分泌系统（下丘脑-垂体-性腺轴） |
| **关联营养元素** | 硒-GPX4/SEPP1、锌（协同精子发生）、NAC/谷胱甘肽（协同抗氧化）、维生素E（膜保护） |
