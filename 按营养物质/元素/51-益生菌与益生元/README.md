# 益生菌与益生元 (益生菌与益生元)

> **维度**: 营养物质/元素
> **关联知识源标签**: 营养-益生菌与益生元

## 常见来源及推荐排序

按补充效果从高到低排序：

1. 酸奶(益生菌)
2. 泡菜(益生菌)
3. 味增(益生菌)
4. 洋葱(益生元)
5. 大蒜(益生元)
6. 香蕉(益生元)
7. 菊苣根(益生元)

## 缺乏或过量影响

- 缺乏: 肠道菌群失调、消化问题、免疫功能下降、炎症增加
- 过量: 腹胀、胀气、腹泻(益生元过量)

## 推荐家常菜系及做法

- 酸奶配浆果
- 泡菜汤
- 味增汤
- 蒜蓉炒菜
- 洋葱汤
