## CAF-001：咖啡因对耐力运动表现的提升——剂量和时间优化

| 字段 | 内容 |
|------|------|
| **源头** | Sports Medicine / Journal of Applied Physiology / ISSN Position Stand |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | Guest NS, et al. "International society of sports nutrition position stand: caffeine and exercise performance." J Int Soc Sports Nutr. 2021;18(1):1. + Ganio MS, et al. "Effect of caffeine on sport-specific endurance performance: a systematic review." J Strength Cond Res. 2009;23(1):315-324. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/33388079/ |
| **知识内容** | 咖啡因是研究最充分的运动助剂之一。ISSN立场声明：①有效剂量3-6 mg/kg体重（约200-400 mg，一杯咖啡约80-100 mg），运动前60分钟摄入；②提升耐力运动表现约2-4%（计时赛、TTT、跑步），效果在持续60分钟以上运动最明显；③低剂量（<3 mg/kg）也有效且副作用更少；④胶囊>咖啡>无水咖啡因，因咖啡中其他成分可能减弱效果；⑤基因型(CYP1A2)和习惯性使用影响个体反应。 |
| **实践意义** | 耐力运动前60分钟摄入3-6 mg/kg咖啡因可提升表现2-4% |
| **关联器官** | 中枢神经系统、肌肉系统 |
| **关联营养元素** | 咖啡因、腺苷受体 |

| 字段 | 内容 |
|------|------|
| **源头** | Journal of Strength and Conditioning Research / European Journal of Applied Physiology |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级（系统综述与Meta分析） |
| **源文献** | Warren GL, et al. "Effect of caffeine ingestion on muscular strength and power: a systematic review and meta-analysis." Med Sci Sports Exerc. 2010;42(7):1375-1387. + Grgic J, et al. "Caffeine ingestion enhances 1RM strength performance: a meta-analysis." J Sci Med Sport. 2018;21(6):558-563. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/20019636/ |
| **知识内容** | 咖啡因不仅提升耐力，还增强力量输出。Warren(2010)系统综述：①咖啡因提升最大自主收缩力量约4-7%；②增强力量耐力（更多重复次数）；③对下肢（深蹲、腿举）效果>上肢。Grgic(2018)Meta分析：1RM力量提升效应量0.20-0.32（虽小但统计显著）。核心机制：①拮抗腺苷A1/A2A受体→增加神经递质释放→增强CNS驱动；②增加肌浆网钙释放→增强兴奋-收缩耦联；③降低疼痛感知→允许更高强度输出。 |
| **实践意义** | 力量训练前60分钟摄入咖啡因可提升1RM约4-7% |
| **关联器官** | 中枢神经系统（运动单位募集）、肌肉系统（兴奋-收缩耦联） |
| **关联营养元素** | 咖啡因、钙离子（兴奋-收缩耦联） |

## CAF-003：CYP1A2基因型对咖啡因代谢和运动反应的影响

| 字段 | 内容 |
|------|------|
| **源头** | Medicine & Science in Sports & Exercise / Nutrients |
| **权威等级** | 🥈 高 |
| **证据等级** | B级（RCT证据，但样本量有限） |
| **源文献** | Womack CJ, et al. "The influence of a CYP1A2 polymorphism on the ergogenic effects of caffeine." J Int Soc Sports Nutr. 2012;9(1):7. + Guest N, et al. "Caffeine, CYP1A2 genotype, and endurance performance in athletes." Med Sci Sports Exerc. 2018;50(8):1570-1578. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/22591721/ |
| **知识内容** | CYP1A2基因编码咖啡因代谢的主要酶。基因型分为：①AA型（"快速代谢者"）——咖啡因代谢快，运动表现获益最大（骑行计时赛提升4.9%）；②AC型（中间型）；③CC型（"慢速代谢者"）——代谢慢，运动表现获益可能减弱甚至反向，且增加高血压和心梗风险。Womack(2012)：AA型运动员10 km骑行计时显著提升，CC型无效果。但后续研究有不一致结果，ISSN认为证据尚不足以推荐常规基因检测指导咖啡因使用。 |
| **实践意义** | 个体对咖啡因反应差异大，CYP1A2基因型可能是因素之一，但暂不推荐常规检测 |
| **关联器官** | 肝脏（CYP1A2代谢）、中枢神经系统 |
| **关联营养元素** | 咖啡因、CYP1A2 |

## CAF-004：咖啡因戒断症状与长期使用的耐受性

| 字段 | 内容 |
|------|------|
| **源头** | Psychopharmacology / DSM-5 |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | Juliano LM, Griffiths RR. "A critical review of caffeine withdrawal: empirical validation of symptoms and signs, incidence, severity, and associated features." Psychopharmacology (Berl). 2004;176(1):1-29. + DSM-5 "Caffeine Withdrawal" (Diagnostic Criteria) |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/15448977/ |
| **知识内容** | 咖啡因戒断是DSM-5正式诊断条目。症状：①头痛（50%，戒断后12-24h开始，持续1-2天）；②疲劳、嗜睡、精力下降；③易怒、焦虑、情绪低落；④注意力不集中、流感样症状。症状在戒断后20-51h达到高峰，持续2-9天。耐受性：持续使用导致腺苷受体上调（部分耐受），但运动表现的助益不会完全消失——习惯性使用者的运动获益虽可能减弱（效应量降低约20-30%），但仍有显著效果。建议：赛前7天停用咖啡因可恢复敏感性。 |
| **实践意义** | 赛前7天戒断可恢复咖啡因敏感性，最大化运动获益 |
| **关联器官** | 中枢神经系统（腺苷受体上调）、脑（戒断性头痛） |
| **关联营养元素** | 咖啡因、腺苷 |

## CAF-005：咖啡因与睡眠——半衰期、腺苷清除与NREM睡眠质量

| 字段 | 内容 |
|------|------|
| **源头** | Sleep Medicine Reviews / Journal of Clinical Sleep Medicine |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | Drake C, et al. "Caffeine effects on sleep taken 0, 3, or 6 hours before going to bed." J Clin Sleep Med. 2013;9(11):1195-1200. + Clark I, Landolt HP. "Coffee, caffeine, and sleep: A systematic review of epidemiological studies and randomized controlled trials." Sleep Med Rev. 2017;31:70-78. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/24235903/ |
| **知识内容** | 咖啡因半衰期约4-6小时（范围2-12h，取决于CYP1A2活性）。关键RCT：睡前6小时摄入400 mg咖啡因（约4杯咖啡）仍显著扰乱睡眠——总睡眠时间减少1小时，睡眠效率降低，NREM深睡眠（慢波睡眠）减少。机制：咖啡因阻断腺苷A1受体→消除睡眠压力积累信号→入睡困难+浅睡。睡前3小时摄入同样有破坏。临床建议：①对睡眠敏感者应在14:00前停止咖啡因；②半衰期长（CC基因型）者更需提前；③下午/晚间运动者需注意运动补剂中咖啡因。 |
| **实践意义** | 咖啡因睡前6小时摄入仍损害睡眠，睡眠敏感者14:00后避免 |
| **关联器官** | 脑（腺苷受体、睡眠-觉醒调节） |
| **关联营养元素** | 咖啡因、腺苷（睡眠压力）、褪黑素 |
