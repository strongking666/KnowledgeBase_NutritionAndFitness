## CA-001：钙+维生素D对骨折和跌倒预防的效果（2026 BMJ最新Meta）

| 字段 | 内容 |
|------|------|
| **源头** | BMJ |
| **权威等级** | 🥇 最高（顶级医学期刊系统综述） |
| **证据等级** | A级（大规模系统综述与Meta分析） |
| **源文献** | Masse O, et al. "Calcium, vitamin D, or combined supplementation to prevent fractures and falls: systematic review and meta-analysis." BMJ. 2026;393:e088050. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/42161415/ |
| **知识内容** | BMJ 2026年最全面的系统综述：钙、维生素D或两者联合补充对骨折和跌倒的预防效果极小或无临床意义。结论挑战了广泛存在的"补钙防骨折"观念。食物来源的钙可能比补充剂更有益。 |
| **实践意义** | 优先通过食物获取钙（牛奶、酸奶、豆腐、芝麻酱、绿叶菜），而非依赖补充剂 |
| **关联器官** | 骨骼与关节（骨密度） |
| **关联营养元素** | 钙+维生素D（协同吸收） |

## CA-002：绝经后女性钙+维生素D补充

| 字段 | 内容 |
|------|------|
| **源头** | UpToDate / 多篇系统综述 |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | Rosen HN. "Calcium and vitamin D supplementation in osteoporosis." UpToDate. Updated Sep 2025. |
| **知识内容** | 对已确诊骨质疏松的患者，建议总钙摄入1200mg/天（饮食+补充剂）+维生素D 800 IU/天。但对于社区居住的饮食充足的成年人，常规补钙不一定必要。钙补充剂的潜在风险包括肾结石和心血管钙化（争议中）。 |
| **实践意义** | 饮食优先；已确诊骨质疏松者才考虑补充剂，需遵医嘱 |
| **关联器官** | 骨骼与关节（骨质疏松）、肾脏（肾结石风险） |
| **关联营养元素** | 钙+维生素D+维生素K2（协同骨代谢） |

## CA-003：钙的食物来源与吸收率

| 字段 | 内容 |
|------|------|
| **源头** | NIH / USDA / AND |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **知识内容** | **钙含量排序**（每100g）：芝麻酱(~975mg) > 奶酪(~700mg) > 小鱼干/虾皮(~500mg) > 豆腐(钙盐凝固, ~350mg) > 酸奶(~120mg) > 牛奶(~120mg) > 羽衣甘蓝(~150mg但吸收率~50%) > 杏仁(~260mg但吸收率较低)。**吸收率**：牛奶中钙吸收率约30%，草酸高的蔬菜（菠菜）仅5-10%。**维生素D**显著提高钙吸收率。 |
| **实践意义** | 每天1杯牛奶/酸奶+1份豆腐+绿叶菜即可满足大部分钙需求 |
| **关联器官** | 骨骼与关节（骨矿化）、消化系统（肠道吸收） |
| **关联营养元素** | 钙+维生素D（吸收促进）+维生素K2（骨钙沉积） |

## 知识源交叉验证

✅ BMJ 2026 + UpToDate + NIH → 食物来源钙优先，补充剂效果有限
✅ 骨质疏松确诊者需要补充剂（1200mg钙+800IU维生素D）
⚠️ 争议：常规补钙是否增加心血管风险——多数证据不支持，但高剂量补充剂应谨慎
