## IOD-001：碘与甲状腺激素合成——TSH-T4-T3轴的营养基础

| 字段 | 内容 |
|------|------|
| **源头** | Endocrine Reviews / Thyroid / Lancet Diabetes & Endocrinology |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级（分子机制完全阐明） |
| **源文献** | Zimmermann MB. "Iodine deficiency." Endocr Rev. 2009;30(4):376-408. + Dunn JT, Dunn AD. "Update on intrathyroidal iodine metabolism." Thyroid. 2001;11(5):407-414. + Brent GA. "Mechanisms of thyroid hormone action." J Clin Invest. 2012;122(9):3035-3043. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/19460960/ |
| **知识内容** | 碘是甲状腺激素的唯一限速底物——无碘则无甲状腺激素：①甲状腺滤泡细胞通过钠碘同向转运体（NIS）主动摄取血碘，甲状腺内碘浓度为血浆的20-50倍——TSH是NIS表达的最强调节因子；②碘的氧化和有机化：甲状腺过氧化物酶（TPO）在H₂O₂存在下将I⁻氧化为活性碘，并将其连接到甲状腺球蛋白（Tg）的酪氨酸残基→生成单碘酪氨酸（MIT）和二碘酪氨酸（DIT）；③偶联反应：TPO催化MIT+DIT→T3（三碘甲状腺原氨酸，活性激素），DIT+DIT→T4（甲状腺素，前体激素）——T4:T3产生比例约14:1；④T4在外周经脱碘酶（DIO1/DIO2，含硒酶）转化为活性T3；⑤反馈调控：T3与下丘脑TRH神经元和垂体促甲状腺激素细胞的甲状腺激素受体（TRβ）结合→抑制TSH分泌——TSH是碘营养状况的最敏感指标；⑥碘摄入急剧变化时，Wolff-Chaikoff效应（急性碘过量→暂时抑制甲状腺激素合成）和Jod-Basedow现象（长期缺碘后补碘→甲亢）反映甲状腺的自我保护机制。 |
| **实践意义** | 碘摄入的稳定性和适量性至关重要；TSH是评估碘营养状态的首选指标；缺碘和碘过量均可导致甲状腺功能障碍 |
| **关联器官** | 甲状腺（滤泡细胞-碘化-激素合成）、垂体（TSH分泌）、下丘脑（TRH） |
| **关联营养元素** | 碘、硒（脱碘酶辅因子-外周T4→T3）、铁（TPO辅因子-血红素）、酪氨酸（底物） |

## IOD-002：全球碘营养状况——从缺碘性甲肿到碘过量

| 字段 | 内容 |
|------|------|
| **源头** | WHO / UNICEF / Iodine Global Network / Lancet |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级（全球流行病学数据+公共卫生干预证据） |
| **源文献** | WHO. "Assessment of iodine deficiency disorders and monitoring their elimination." 3rd ed. 2007. + Zimmermann MB, Andersson M. "Global endocrinology: Global perspectives in endocrinology: coverage of iodized salt programs and iodine status in 2020." Eur J Endocrinol. 2021;185(1):R13-R21. + Farebrother J, et al. "Excess iodine intake: sources, assessment, and effects on thyroid function." Ann N Y Acad Sci. 2019;1446(1):44-65. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/33989173/ |
| **知识内容** | 全球碘营养状况的双重挑战——缺乏与过量并存：①全球约20亿人碘摄入不足，其中约5000万出现临床症状——碘缺乏仍是全球可预防智力障碍的首要原因；②碘盐全球覆盖率约88%，但仍有21个国家碘营养不足（2020年数据）——主要在撒哈拉以南非洲、南亚和部分欧洲国家；③中国自1995年实施全民食盐加碘（USI）后，缺碘性甲状腺肿（俗称大脖子病）从20%+降至<5%，但部分地区出现碘过量问题；④碘过量的来源：饮用水（高碘地区，中国11个省份有高碘水问题）、海藻类补充剂、含碘消毒剂/造影剂、胺碘酮（含碘抗心律失常药）；⑤碘过量临床后果：亚临床甲减（TSH↑，T4正常）、自身免疫性甲状腺炎风险增加、甲状腺乳头状癌（PTC）发病率上升（但死亡率未增，部分为检出率增加）；⑥WHO推荐尿碘中位数（MUIC）评估人群碘状态：<100μg/L=不足，100-199μg/L=充足，200-299μg/L=超足量，≥300μg/L=过量。 |
| **实践意义** | 使用碘盐是最经济有效的补碘策略；避免盲目额外补碘（尤其高碘地区）；育龄女性应关注碘状态 |
| **关联器官** | 甲状腺（滤泡-自身免疫-结节）、大脑与神经系统（胎儿-新生儿发育） |
| **关联营养元素** | 碘、硒（抗氧化保护甲状腺）、铁（影响碘利用效率） |

## IOD-003：孕期碘需求增加及对胎儿神经发育的影响

| 字段 | 内容 |
|------|------|
| **源头** | Lancet / Journal of Clinical Endocrinology & Metabolism / WHO |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级（观察性证据强；RCT因伦理限制有限但机制明确） |
| **源文献** | Bath SC, et al. "Effect of inadequate iodine status in UK pregnant women on cognitive outcomes in their children: results from the ALSPAC study." Lancet. 2013;382(9889):331-337. + Zimmermann MB. "The effects of iodine deficiency in pregnancy and infancy." Paediatr Perinat Epidemiol. 2012;26(Suppl 1):108-117. + WHO. "Guideline: fortification of food-grade salt with iodine for the prevention and control of iodine deficiency disorders." 2014. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/23706508/ |
| **知识内容** | 孕期是碘需求最关键的窗口期：①孕期碘需求增加50%（RDA: 220μg/天 vs 非孕期150μg/天，WHO建议250μg/天）：肾碘清除率增加（GFR↑）+胎儿和胎盘碘摄取+T4产量增加+甲状腺结合球蛋白（TBG）升高（雌激素↑）；②胎儿甲状腺在孕10-12周才开始摄碘——此前胎儿神经发育完全依赖母体T4跨胎盘转运；③Bath等里程碑ALSPAC研究（1,040名英国孕妇）：孕期碘不足（尿碘<150μg/g肌酐）母亲的子女8岁时语言智商低3.2分，阅读准确性和理解力均差；④即使轻中度碘缺乏也影响后代认知——"碘缺乏导致的智力损伤谱系"从克汀病（重度，IQ<30）到轻度认知下降；⑤克汀病分两型：神经型（不可逆，宫内T4不足导致大脑皮层和耳蜗损伤）和黏液水肿型（可部分纠正，出生后甲状腺功能衰竭+缺碘）；⑥孕期碘补充RCT：从孕早期补充碘（150-200μg/天）可改善后代神经发育评分，但证据级别受限于伦理（不能设缺碘对照组）。 |
| **实践意义** | 所有育龄和孕期女性应确保碘充足（使用碘盐+含碘 prenatal 补充剂）；孕前即应开始关注；轻度缺碘已影响后代IQ |
| **关联器官** | 大脑与神经系统（胎儿大脑皮层-海马-耳蜗）、甲状腺（孕期代偿性增大）、胎盘（T4转运） |
| **关联营养元素** | 碘、硒（孕期需求同时增加）、铁（孕期需求增加，协同影响甲状腺）、叶酸 |

## IOD-004：海藻类食物的碘含量差异及安全食用量

| 字段 | 内容 |
|------|------|
| **源头** | British Journal of Nutrition / Thyroid / EFSA / 食品安全风险评估 |
| **权威等级** | 🥇 最高 |
| **证据等级** | B级（含量测定数据充分，但安全指南多为专家共识） |
| **源文献** | Teas J, et al. "Variability of iodine content in common commercially available edible seaweeds." Thyroid. 2004;14(10):836-841. + EFSA. "Scientific Opinion on Dietary Reference Values for iodine." EFSA Journal. 2014;12(5):3660. + Bouga M, Combet E. "Emergence of seaweed and seaweed-containing foods in the UK: focus on labeling, iodine content, toxicity and nutrition." Foods. 2015;4(2):240-253. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/24844570/ |
| **知识内容** | 海藻类食物的碘含量差异巨大——从营养补充到中毒风险：①不同海藻碘含量（μg/g干重）：昆布/Kombu（海带）2353-10,104μg/g——1g即可能超过UL；羊栖菜/Hijiki 391-630μg/g；裙带菜/Wakame 42-420μg/g；紫菜/Nori 16-55μg/g——最安全；②昆布1片（约2g干重）可含碘4,700-20,000μg——远超UL（1100μg/天，成人）和WHO推荐单次不超过500μg；③日本传统高碘饮食（日均1-3mg碘，为WHO推荐的10-30倍）——日本甲状腺对高碘的适应（Wolff-Chaikoff效应逃逸机制上调）但仍有部分人出现碘诱导甲减；④海藻碘的生物利用度差异：昆布碘生物利用度>90%，但部分海藻含碘结合纤维降低吸收；⑤安全食用建议：紫菜（Nori）每日可食10-30g；裙带菜（Wakame）每日≤5g（干重）；昆布/海带（Kombu）不建议日常食用——仅作为调味料微量使用（每次<0.5g）；⑥孕妇和甲状腺自身免疫疾病者应避免高碘海藻。 |
| **实践意义** | 海藻补充碘需严格区分种类和量——紫菜安全，昆布/海带极易过量；不要将海藻作为日常补碘唯一来源 |
| **关联器官** | 甲状腺（碘负荷-Wolff-Chaikoff效应）、消化系统（碘吸收） |
| **关联营养元素** | 碘、硒（协同甲状腺保护，高碘时需确保硒充足） |
