## SOD-001：钠摄入与血压——从DASH试验到PURE研究的争议

| 字段 | 内容 |
|------|------|
| **源头** | NEJM / The Lancet / 多项RCT与前瞻性队列 |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | Sacks FM, et al. "Effects on blood pressure of reduced dietary sodium and the Dietary Approaches to Stop Hypertension (DASH) diet." N Engl J Med. 2001;344:3-10. + O'Donnell M, et al. "Urinary sodium and potassium excretion, mortality, and cardiovascular events." N Engl J Med. 2014;371:612-623. (PURE Study) |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/11136953/ |
| **知识内容** | DASH试验证实低钠饮食降低血压（钠从150 mmol/d降至100 mmol/d，收缩压降低5.5 mmHg），且低钠+DASH饮食效果叠加（收缩压降11.5 mmHg）。但PURE研究（101,945人）发现钠摄入与心血管事件的J型曲线：极低钠（<3 g/d）和高钠（>6 g/d）均增加风险，最低风险在4-5 g/d。争议核心：公共卫生普适性限制 vs 个体化钠摄入目标。 |
| **实践意义** | 适度限钠（4-5 g/d）有益，极端限钠可能对部分人群有害 |
| **关联器官** | 心脏、血管系统、肾脏 |
| **关联营养元素** | 钠、钾（钠钾比）、钙、镁 |

## SOD-002：运动出汗的钠丢失量及运动饮料补钠策略

| 字段 | 内容 |
|------|------|
| **源头** | Journal of Sports Sciences / Medicine & Science in Sports & Exercise |
| **权威等级** | 🥈 高 |
| **证据等级** | B级 |
| **源文献** | Shirreffs SM, Maughan RJ. "Whole body sweat collection in humans: an improved method with preliminary data on electrolyte content." J Appl Physiol. 1997;82:336-341. + ACSM Position Stand on Exercise and Fluid Replacement (2007) |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/9029232/ |
| **知识内容** | 汗液钠浓度个体差异极大（10-90 mmol/L），同一人不同条件也波动。1小时中等强度运动可丢失钠0.5-2.0 g。热适应后汗钠浓度降低（醛固酮增加钠重吸收）。运动饮料钠含量通常10-25 mmol/L（约230-575 mg/L），不足以完全替代丢失量，但可促进水分吸收和保留。关键策略：①了解个人出汗率和汗钠浓度；②>2小时运动需主动补钠；③运动后餐食补钠比饮料更有效。 |
| **实践意义** | 运动补钠需个体化，依赖运动时长和出汗率，日常餐食是主渠道 |
| **关联器官** | 肌肉系统（痉挛预防）、肾脏（钠重吸收调节） |
| **关联营养元素** | 钠、钾、氯、水分 |

## SOD-003：加工食品作为隐形钠来源——中国居民钠摄入构成

| 字段 | 内容 |
|------|------|
| **源头** | 中国居民营养与慢性病状况报告 / BMJ / WHO |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | 中国居民营养与慢性病状况报告（2020年）. 国务院新闻办公室. + He FJ, et al. "Salt reduction in China: a state-of-the-art review." BMJ. 2022;376:e067944. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/35387736/ |
| **知识内容** | 中国居民人均钠摄入约10.5 g/d（相当于盐26.3 g/d），是世界卫生组织推荐上限（5 g/d盐）的5倍。钠来源构成：①烹调盐占约70-75%（家庭+餐饮）；②加工食品占约15-20%（酱油、味精、腌制品、方便面）；③天然食物仅占<5%。关键认知偏差：公众对"隐形钠"（酱油、蚝油、鸡精、挂面、面包）缺乏意识。减钠策略：用钾盐替代部分钠盐（中国低钠盐推广RCT证实有效降血压并减少心血管事件）。 |
| **实践意义** | 减少烹调盐+识别隐形钠来源（调味品、加工食品）是减钠关键 |
| **关联器官** | 心脏、血管系统、肾脏 |
| **关联营养元素** | 钠、钾（钠钾替代）、碘（碘盐问题） |

## SOD-004：低钠血症——耐力运动的隐形杀手

| 字段 | 内容 |
|------|------|
| **源头** | NEJM / Clinical Journal of Sport Medicine |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | Almond CS, et al. "Hyponatremia among runners in the Boston Marathon." N Engl J Med. 2005;352:1550-1556. + Hew-Butler T, et al. "Statement of the Third International Exercise-Associated Hyponatremia Consensus Development Conference." Clin J Sport Med. 2015;25(4):303-320. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/15829535/ |
| **知识内容** | 2002年波士顿马拉松中13%参赛者有低钠血症（血钠<135 mmol/L），0.6%为严重低钠血症（<120 mmol/L，可致死）。根本原因不是缺钠，而是过量饮水导致的稀释性低钠血症。危险因素：①运动时间>4小时；②体重增加（水中毒）；③女性；④NSAIDs使用。核心预防原则：渴了就喝，不要强迫性补水；>4小时运动使用含钠运动饮料而非纯水。 |
| **实践意义** | 耐力运动"按需饮水"而非"过量补水"，>4小时运动避免纯水 |
| **关联器官** | 脑（脑水肿风险）、肾脏（水利尿调节） |
| **关联营养元素** | 钠、水分、ADH（抗利尿激素） |
