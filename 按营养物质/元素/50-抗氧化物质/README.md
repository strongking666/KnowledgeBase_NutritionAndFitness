# 抗氧化物质 (抗氧化物质)

> **维度**: 营养物质/元素
> **关联知识源标签**: 营养-抗氧化物质

## 常见来源及推荐排序

按补充效果从高到低排序：

1. 蓝莓(花青素)
2. 绿茶(儿茶素)
3. 黑巧克力(黄酮类)
4. 番茄(番茄红素)
5. 姜黄(姜黄素)
6. 红酒(白藜芦醇-适量)
7. 石榴

## 缺乏或过量影响

- 缺乏: 氧化应激增加、慢性炎症、衰老加速、慢性病风险增加
- 过量: 补充剂过量可能促氧化(悖论)

## 推荐家常菜系及做法

- 蓝莓酸奶
- 绿茶
- 番茄炖牛腩
- 姜黄牛奶
- 石榴沙拉
