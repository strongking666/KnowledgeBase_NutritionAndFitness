# 锌

> **模块**: 43-锌（营养维度）
> **标签**: 营养-锌 | 器官-免疫系统 | 器官-皮肤 | 器官-肌肉系统 | 器官-内分泌系统
> **最后更新**: 2026-06-30

---

## ZN-001：锌与免疫功能

| 字段 | 内容 |
|------|------|
| **源头** | NIH / WHO / 多项系统综述 |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | Multiple NIH reviews + WHO guidelines on zinc |
| **知识内容** | 锌是300+种酶的辅因子，对免疫细胞发育和功能至关重要。**缺乏后果**：T细胞功能下降、胸腺萎缩、伤口愈合延迟、感染易感性增加（尤其呼吸道和消化道感染）。WHO推荐腹泻儿童补锌。**推荐摄入量**：成年男性11mg/天，女性8mg/天。 |
| **实践意义** | 锌摄入不足时（素食者、运动员出汗多），优先食补（牡蛎、红肉、南瓜籽） |
| **关联器官** | 免疫系统（免疫细胞功能）、皮肤（伤口愈合）、内分泌系统（睾酮合成） |
| **关联营养元素** | 锌-牡蛎（最高来源） |

## ZN-002：锌与运动/肌肉

| 字段 | 内容 |
|------|------|
| **源头** | ISSN / 多项RCT |
| **权威等级** | 🥈 高 |
| **证据等级** | B级 |
| **源文献** | ISSN position statements + multiple RCTs |
| **知识内容** | 运动增加锌流失（汗液、尿液）。运动员（尤其耐力运动员）锌缺乏风险增加。锌参与蛋白质合成和睾酮产生，对肌肉恢复和生长有支持作用。但补充锌超过RDA对运动表现的额外益处证据不足。锌与铜竞争吸收——长期高剂量补锌可能导致铜缺乏。 |
| **实践意义** | 运动员确保饮食锌充足即可，无明确证据支持额外补充提升表现 |
| **关联器官** | 肌肉系统（蛋白质合成）、内分泌系统（睾酮） |
| **关联营养元素** | 锌（注意：过量抑制铜吸收） |

## ZN-003：锌的食物来源与吸收

| 字段 | 内容 |
|------|------|
| **源头** | NIH Office of Dietary Supplements |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **知识内容** | **锌含量排序**（每100g）：牡蛎(~78mg) >> 红肉/牛肉(~5mg) > 蟹/虾(~3-4mg) > 南瓜籽(~7mg但吸收率受植酸影响) > 腰果(~5mg) > 豆类(~1-3mg) > 全谷物(~1-3mg)。**吸收率**：动物来源锌吸收率(~20-40%) > 植物来源(~10-20%，植酸干扰)。**增强吸收**：动物蛋白、发酵（减少植酸）。**抑制吸收**：高植酸食物（全谷物/豆类未浸泡处理）、高钙同时摄入。 |
| **实践意义** | 每周吃1-2次牡蛎或红肉即可满足锌需求；素食者需注意浸泡/发酵豆类和谷物 |
| **关联器官** | 消化系统（肠道吸收）、免疫系统 |
| **关联营养元素** | 锌-动物来源优先（牡蛎>红肉>南瓜籽） |

## 知识源交叉验证

✅ NIH + WHO → 锌缺乏在发展中国家常见，影响免疫和生长
✅ 运动员锌流失增加，但补充效果证据不足
✅ 牡蛎是自然界锌含量最高的食物
