# 维生素B族 (维生素B族)

> **维度**: 营养物质/元素
> **关联知识源标签**: 营养-维生素B族

## 常见来源及推荐排序

按补充效果从高到低排序：

1. 全谷物(B1/B3)
2. 瘦肉(B1/B3/B6/B12)
3. 鸡蛋(B2/B7/B12)
4. 豆类(B1/B9)
5. 深绿叶菜(B9)
6. 动物肝脏(B12)
7. 乳制品(B2/B12)

## 缺乏或过量影响

- B1缺乏: 脚气病、神经损伤
- B9缺乏: 巨幼细胞贫血、神经管缺陷(孕妇)
- B12缺乏: 恶性贫血、神经病变、认知障碍

## 推荐家常菜系及做法

- 全麦面包
- 水煮蛋
- 菠菜沙拉
- 瘦肉粥
- 牛奶
- 豆类炖汤
