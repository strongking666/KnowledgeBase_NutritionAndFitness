## VITD-002：维生素D对骨骼健康的影响（IPD Meta分析）

| 字段 | 内容 |
|------|------|
| **源头** | American Journal of Clinical Nutrition |
| **权威等级** | 🥇 最高（个体参与者数据Meta分析） |
| **证据等级** | A级（11项RCT，1439名参与者） |
| **源文献** | Wu F, et al. "Vitamin D supplementation for improving bone density in vitamin D-deficient children and adolescents." Am J Clin Nutr. 2023;118(3):498-506. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/37661104/ |
| **知识内容** | 维生素D补充对髋部骨密度有小幅正面效果。但基线维生素D水平与补充效果无明显交互作用——即使基线不足者，1年补充对骨密度的改善也有限。结论：对健康儿童青少年的骨密度，维生素D补充的临床获益有限。 |
| **实践意义** | 单独维生素D对骨密度的改善有限，需配合钙+运动 |
| **关联器官** | 骨骼与关节（骨密度） |
| **关联营养元素** | 维生素D+钙（协同作用） |

## VITD-003：维生素D与免疫功能

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / Nutrients / 多项系统综述 |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | 多项Meta分析综合（2024-2025） |
| **知识内容** | 维生素D在先天和适应性免疫中发挥关键作用：①维生素D受体存在于几乎所有免疫细胞；②促进抗菌肽（cathelicidin）产生，增强先天免疫；③调节T细胞分化，抑制过度炎症；④观察性研究一致显示低维生素D水平与感染风险增加相关；⑤RCT证据：补充维生素D可减少呼吸道感染（尤其基线不足者）。 |
| **实践意义** | 维持维生素D充足状态（>30 ng/mL）有助于免疫功能 |
| **关联器官** | 免疫系统（免疫调节） |
| **关联营养元素** | 维生素D-免疫调节 |
