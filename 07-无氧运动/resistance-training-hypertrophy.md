## RT-001：训练频率对肌肥大的影响

| 字段 | 内容 |
|------|------|
| **源头** | Sports Medicine / PubMed |
| **权威等级** | 🥇 最高（系统综述与Meta分析） |
| **证据等级** | A级 |
| **源文献** | Schoenfeld BJ, et al. "Effects of Resistance Training Frequency on Measures of Muscle Hypertrophy: A Systematic Review and Meta-Analysis." Sports Med. 2016;46:1689-1697. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/27102172/ |
| **知识内容** | 在等量训练量下，每周训练每个肌群2次的肌肥大效果优于每周1次（效应量0.49 vs 0.30, p=0.002）。每周3次是否优于2次尚无定论。 |
| **实践意义** | 每个肌群每周至少训练2次，最大化肌肥大 |
| **关联器官** | 肌肉系统（肌纤维肥大） |
| **关联营养元素** | 蛋白质（肌肉蛋白合成-MPS） |

## RT-002：训练量-频率的剂量-反应关系（最新Meta回归）

| 字段 | 内容 |
|------|------|
| **源头** | SportRxiv / 多中心Meta回归 |
| **权威等级** | 🥇 最高（Meta回归，67项研究，2,058人） |
| **证据等级** | A级 |
| **源文献** | Pelland J, et al. "The Resistance Training Dose-Response: Meta-Regressions Exploring the Effects of Weekly Volume and Frequency on Muscle Hypertrophy and Strength Gain." SportRxiv. 2025. |
| **URL** | https://doi.org/10.51224/SRXIV.460 |
| **知识内容** | 训练量增加与肌肥大/力量增益正相关（100%概率>0），但存在收益递减效应。力量增益的收益递减比肌肥大更明显。频率对肌肥大的独立效应不明确（<100%概率），但对力量增益有正效应。关键发现：区分"直接组数"和"间接组数"对预测训练效果至关重要。 |
| **实践意义** | 优先增加高质量直接组数（每周10-20组/肌群），再考虑频率优化 |
| **关联器官** | 肌肉系统 |
| **关联营养元素** | 蛋白质（支持恢复与合成） |

## RT-003：ACSM抗阻训练进阶模型

| 字段 | 内容 |
|------|------|
| **源头** | ACSM Position Stand |
| **权威等级** | 🥇 最高（专业学会立场声明） |
| **证据等级** | A级（数百项RCT基础） |
| **源文献** | American College of Sports Medicine. "Progression Models in Resistance Training for Healthy Adults." Med Sci Sports Exerc. 2009;41(3):687-708. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/19204579/ |
| **知识内容** | **新手**：每周2-3次全身训练，1-3组/动作，8-12RM。**中级**：每周4次上下肢分化，多组数。**高级**：每周4-6次，分化训练，周期化。**肌肥大**：1-3组×8-12RM，组间休息1-2分钟。**力量**：2-6组×1-6RM，组间休息2-3分钟。**肌耐力**：≤2组×15-25RM。 |
| **实践意义** | 根据训练目标和经验水平选择合适的组数/次数/频率 |
| **关联器官** | 肌肉系统、骨骼与关节（骨骼健康）、内分泌系统（激素反应） |
| **关联营养元素** | 蛋白质（恢复合成）、肌酸（ATP-PC系统） |

## RT-004：老年人抗阻训练量与功能改善（2025网络Meta）

| 字段 | 内容 |
|------|------|
| **源头** | Sports Medicine |
| **权威等级** | 🥇 最高（网络Meta分析，151项RCT） |
| **证据等级** | A级 |
| **源文献** | Radaelli R, et al. "Effects of Resistance Training Volume on Physical Function, Lean Body Mass and Lower-Body Muscle Hypertrophy and Strength in Older Adults." Sports Med. 2025;55(1):167-192. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/39405023/ |
| **知识内容** | 151项RCT网络Meta分析：较高训练量（每周≥21组/肌群）对老年人瘦体重增加和下肢肌力提升效果最大，但也存在收益递减。中等训练量（10-20组/周）在功能改善方面效率最高。 |
| **实践意义** | 老年人推荐中等训练量（每周10-20组/肌群），平衡效果与安全 |
| **关联器官** | 肌肉系统（肌少症预防）、骨骼与关节（骨密度） |
| **关联营养元素** | 蛋白质（老年人需要更高蛋白摄入）、维生素D（肌力支持） |

✅ 训练量是关键驱动因素，存在收益递减
✅ 老年人抗阻训练安全且必要
