## 知识条目

### PRO-001：蛋白质RDA基准值

| 字段 | 内容 |
|------|------|
| **源头** | WHO / NIH / USDA Dietary Reference Intakes |
| **权威等级** | 🥇 最高（国际机构指南） |
| **证据等级** | A级（Meta分析基础） |
| **源文献** | Rand WM, et al. "Meta-analysis of nitrogen balance studies for estimating protein requirements in healthy adults." Am J Clin Nutr. 2003;77(1):109-127. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/12499330/ |
| **知识内容** | 健康成年人蛋白质RDA（推荐膳食摄入量）为 **0.8 g/kg体重/天**。此值基于氮平衡研究的Meta分析，可满足97.5%健康成年人的基本需求。注意：这是防止缺乏的最低量，不是最优量。 |
| **注意事项** | RDA适用于久坐健康成人；运动员、老年人、减重人群需要更高摄入 |

### PRO-002：蛋白质高于RDA对瘦体重的影响

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / Advances in Nutrition |
| **权威等级** | 🥇 最高（系统综述与Meta分析） |
| **证据等级** | A级（Meta分析，18项RCT，22组比较） |
| **源文献** | Hudson JL, et al. "Protein Intake Greater than the RDA Differentially Influences Whole-Body Lean Mass Responses to Purposeful Catabolic and Anabolic Stressors: A Systematic Review and Meta-analysis." Adv Nutr. 2020;11(3):548-558. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/31794597/ |
| **知识内容** | 当处于分解代谢压力（如热量限制减重）时，蛋白质摄入高于RDA（>0.8g/kg）可显著减少瘦体重损失（WMD: 0.36 kg, 95%CI: 0.06-0.67）。在抗阻训练（合成代谢压力）期间，高于RDA的蛋白质摄入可增加瘦体重（WMD: 0.77 kg, 95%CI: 0.23-1.31）。在无压力状态下（不节食也不训练），高于RDA无额外好处。 |
| **实践意义** | 减重期建议蛋白质 ≥1.2-1.6g/kg；增肌期建议 1.6-2.2g/kg |

### PRO-003：蛋白质摄入对运动表现的影响

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / Frontiers in Nutrition |
| **权威等级** | 🥇 最高（系统综述与Meta分析） |
| **证据等级** | A级（28项RCT，373名运动员） |
| **源文献** | "The effect of protein intake on athletic performance: a systematic review and meta-analysis." Front Nutr. 2024;11:1455728. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/39628467/ |
| **知识内容** | 蛋白质摄入对运动表现整体改善不显著（SMD=0.12），但对耐力表现有显著改善（SMD=0.17），对肌肉糖原水平有显著提高（SMD=0.74）。蛋白质+碳水共摄入比单独高蛋白更有效（SMD=0.36）。 |
| **实践意义** | 耐力运动员应关注蛋白质+碳水的组合摄入，而非单纯高蛋白 |

### PRO-004：蛋白质摄入时机

| 字段 | 内容 |
|------|------|
| **源头** | PubMed / Nutrients |
| **权威等级** | 🥇 最高（系统综述与Meta分析） |
| **证据等级** | A级 |
| **源文献** | Casuso RA, Goossens L. "Does Protein Ingestion Timing Affect Exercise-Induced Adaptations? A Systematic Review with Meta-Analysis." Nutrients. 2025;17(13):2070. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/40647175/ |
| **知识内容** | 蛋白质摄入时机（运动前/中/后）对肌肉力量和肌肥大的影响，在总蛋白质摄入量充足的情况下差异不显著。总日蛋白质摄入量比精确的摄入时机更重要。但运动后2小时内摄入蛋白质可能对恢复有边际效益。 |
| **实践意义** | 优先确保每日总蛋白质达标，时机为次要因素 |

## 争议与说明

| 议题 | 观点A | 观点B | 评估 |
|------|-------|-------|------|
| 总脂肪摄入限制 | WHO建议总脂肪<30%能量 | Harvard营养系认为低脂饮食无额外益处 | 争议主要在于总脂肪量，一致同意减少饱和脂肪和反式脂肪 |
| 蛋白质RDA是否足够 | RDA 0.8g/kg满足基本需求 | 多数运动营养机构建议1.2-2.2g/kg | RDA是"最低需求"非"最优摄入"，不同人群需求不同 |
