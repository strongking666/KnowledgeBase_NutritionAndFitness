### FIB-001：膳食纤维推荐摄入量

| 字段 | 内容 |
|------|------|
| **源头** | Dietary Guidelines for Americans 2025-2030 / WHO / NIH |
| **权威等级** | 🥇 最高（政府膳食指南） |
| **证据等级** | A级（大型队列研究+Meta分析基础） |
| **源文献** | Dietary Guidelines for Americans 2025-2030, USDA/HHS; McKeown NM, et al. BMJ. 2022;378:e054370 |
| **URL** | https://realfood.gov / https://pmc.ncbi.nlm.nih.gov/articles/PMC9298262/ |
| **知识内容** | **成年人推荐量**：每1000kcal摄入14g纤维。男性约28-38g/天，女性约22-25g/天。WHO建议每日至少25g纤维。中国居民膳食指南建议25-30g/天。全球多数人群实际摄入量仅为推荐量的一半。 |
| **注意事项** | 增加纤维摄入应循序渐进，同时增加饮水量，避免腹胀 |

### FIB-002：膳食纤维类型与功能

| 字段 | 内容 |
|------|------|
| **源头** | AND Position Paper / NIH MedlinePlus / CDC |
| **权威等级** | 🥇 最高（专业学会立场声明+政府机构） |
| **证据等级** | A级 |
| **源文献** | AND Position: Health Implications of Dietary Fiber. J Acad Nutr Diet. 2015;115(11):1861-1870. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/26514720/ |
| **知识内容** | **可溶性纤维**：溶于水形成凝胶，减慢消化，帮助控制血糖和胆固醇。来源：燕麦、苹果、香蕉、豆类、牛油果。**不可溶性纤维**：不溶于水，增加粪便体积，促进肠道蠕动，提高胰岛素敏感性。来源：全麦、麸皮、坚果、种子、果蔬皮。两类纤维均对健康重要，应同时摄入。 |
| **实践意义** | 多样化纤维来源优于单一补充剂 |

### FIB-003：膳食纤维的健康益处（循证）

| 字段 | 内容 |
|------|------|
| **源头** | BMJ / Lancet / WHO / World Cancer Research Fund |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级（多项Meta分析） |
| **源文献** | McKeown NM, et al. BMJ. 2022; Reynolds A, et al. Lancet. 2019;393:434-445 |
| **URL** | https://pmc.ncbi.nlm.nih.gov/articles/PMC9298262/ |
| **知识内容** | 高纤维饮食与以下健康结局显著相关：①全因死亡率降低15-30%；②心血管疾病风险降低；③2型糖尿病风险降低；④结直肠癌风险降低（WCRF确认）；⑤体重管理改善（增加饱腹感）；⑥肠道微生物组多样性增加，支持免疫功能；⑦便秘和憩室病减少。剂量-反应关系：每增加8g/天纤维，全因死亡风险降低5-10%。 |
| **实践意义** | 纤维摄入是少数有明确剂量-反应关系证据的营养干预之一 |

### FIB-004：纤维补充剂 vs 食物来源

| 字段 | 内容 |
|------|------|
| **源头** | Harvard T.H. Chan School of Public Health / BMJ |
| **权威等级** | 🥈 高（顶级学术机构） |
| **证据等级** | B级 |
| **源文献** | Harvard Health Publishing. "The facts on fiber." 2024 |
| **URL** | https://www.health.harvard.edu/diet-and-nutrition/the-facts-on-fiber |
| **知识内容** | 优先从食物中获取纤维，而非补充剂。食物提供维生素、矿物质和植物化学物，补充剂无法替代。仅在以下情况考虑补充剂：①食物摄入困难（咀嚼问题）；②药物引起的便秘；③特殊低碳水饮食。常见品牌：Metamucil（洋车前子）、Citrucel（甲基纤维素）、Benefiber（小麦糊精）。补充剂应配合充足饮水。 |
| **实践意义** | 食物优先原则；补充剂为辅助手段 |
