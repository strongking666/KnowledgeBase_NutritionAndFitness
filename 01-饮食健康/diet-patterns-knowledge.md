## DPT-001：地中海饮食的循证综述——从PREDIMED到长期随访

| 字段 | 内容 |
|------|------|
| **源头** | NEJM / The Lancet / Cochrane |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | Estruch R, et al. "Primary Prevention of Cardiovascular Disease with a Mediterranean Diet Supplemented with Extra-Virgin Olive Oil or Nuts." N Engl J Med. 2018;378(25):e34. (PREDIMED retracted and republished) + Guasch-Ferré M, et al. "Mediterranean diet and cardiovascular health: Teachings of the PREDIMED study." Adv Nutr. 2014;5(3):330S-336S. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/29897866/ |
| **知识内容** | PREDIMED试验（7447人，中位随访4.8年）是地中海饮食最核心的RCT：地中海饮食+特级初榨橄榄油组主要心血管事件（心梗、卒中、心血管死亡）降低31% (HR 0.69)；地中海饮食+坚果组降低28% (HR 0.72)。长期随访（PREDIMED-Plus，2022）：地中海饮食+能量限制+体力活动进一步减少心血管事件。核心成分：①特级初榨橄榄油（主脂肪源）；②每日坚果；③蔬果/豆类/全谷物；④每周≥3次鱼类；⑤限制红肉和加工食品；⑥可选红酒（适度）。 |
| **实践意义** | 地中海饮食是证据最强的心血管保护饮食模式 |
| **关联器官** | 心脏、血管系统 |
| **关联营养元素** | 单不饱和脂肪酸（橄榄油）、omega-3（鱼类）、多酚、坚果 |

## DPT-002：DASH饮食降血压机制——多营养协同

| 字段 | 内容 |
|------|------|
| **源头** | NEJM / Hypertension |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级 |
| **源文献** | Appel LJ, et al. "A clinical trial of the effects of dietary patterns on blood pressure." N Engl J Med. 1997;336:1117-1124. (DASH Trial) + Sacks FM, et al. "Effects on blood pressure of reduced dietary sodium and the DASH diet." N Engl J Med. 2001;344:3-10. (DASH-Sodium) |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/9099655/ |
| **知识内容** | DASH饮食核心特征：高水果/蔬菜/低脂乳制品，全谷物，禽/鱼/坚果，限制红肉/甜食/含糖饮料。DASH试验结果：DASH饮食组收缩压降低5.5 mmHg，舒张压降低3.0 mmHg（vs对照）；高血压亚组效果更大（-11.4/-5.5 mmHg）。DASH-Sodium叠加试验：DASH饮食+低钠（100 mmol/d）使收缩压额外降低3.0 mmHg。降血压机制是多营养协同而非单一因素：高钾（>4.7 g/d）促进钠排泄和血管扩张、高钙和镁协同血管舒张、低钠减少容量负荷、高膳食硝酸盐（蔬菜）→NO通路。 |
| **实践意义** | DASH饮食通过多营养协同降血压，效果堪比单药治疗 |
| **关联器官** | 心脏、血管系统、肾脏 |
| **关联营养元素** | 钾、钙、镁、钠（限制）、膳食硝酸盐 |

## DPT-003：中国居民膳食指南2022关键变化——从"多吃"到"合理搭配"

| 字段 | 内容 |
|------|------|
| **源头** | 中国营养学会 |
| **权威等级** | 🥇 最高（国家指南） |
| **证据等级** | A级（基于系统综述与Meta分析） |
| **源文献** | 中国营养学会.《中国居民膳食指南（2022）》. 人民卫生出版社, 2022. |
| **URL** | https://www.cnsoc.org/ |
| **知识内容** | 2022版膳食指南关键变化（vs 2016版）：①盐摄入上限从<6 g/d收紧至<5 g/d（与WHO一致）；②增加"足量饮水"推荐（男性1.7 L/d，女性1.5 L/d）；③强调全谷物（每日50-150 g），从"多吃"到"合理搭配"；④首次纳入"公筷分餐"和"会烹会选"（烹饪技能）；⑤新增"东方健康膳食模式"（基于江浙沪粤等地区长寿人群饮食特点）。核心转变：从单纯营养摄入量推荐→整体饮食模式和食物选择行为指导。 |
| **实践意义** | 中国膳食指南从"量"到"质+模式"转变，盐<5 g/d，强调全谷物 |
| **关联器官** | 全身系统性 |
| **关联营养元素** | 全谷物、钠（盐<5 g/d）、膳食纤维、水 |

## DPT-004：超加工食品的健康风险——NOVA分类与前瞻性队列

| 字段 | 内容 |
|------|------|
| **源头** | BMJ / The Lancet Public Health / JAMA Internal Medicine |
| **权威等级** | 🥇 最高 |
| **证据等级** | A级（大规模前瞻性队列+Meta分析） |
| **源文献** | Lane MM, et al. "Ultra-processed food exposure and adverse health outcomes: umbrella review of epidemiological meta-analyses." BMJ. 2024;384:e077310. + Srour B, et al. "Ultra-processed food intake and risk of cardiovascular disease: prospective cohort study (NutriNet-Santé)." BMJ. 2019;365:l1451. |
| **URL** | https://pubmed.ncbi.nlm.nih.gov/38418082/ |
| **知识内容** | 2024年BMJ伞状综述（45项Meta分析，近1000万人）提供了超加工食品(UPF)风险的迄今最强证据：①UPF高摄入与32种不良健康结局中的28种显著相关；②全因死亡率HR 1.21，心血管死亡率HR 1.50，2型糖尿病HR 1.40；③焦虑和常见精神障碍风险增加48-53%。NOVA分类第4组（UPF）定义：配方中含很少或没有完整食物，使用工业添加剂（乳化剂、人造甜味剂、色素等）。机制：高能量密度、低饱腹感、促炎成分、扰乱肠道菌群、置换未加工食物。 |
| **实践意义** | 减少超加工食品是改善健康最直接的饮食改变之一 |
| **关联器官** | 全身系统性、肠道（菌群）、代谢系统 |
| **关联营养元素** | 膳食纤维（UPF缺乏）、添加糖、乳化剂 |
